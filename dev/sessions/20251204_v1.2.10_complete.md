**Commits:** 9107fee1

**File:** `extensions/vscode-udos/src/commands/knowledge-checker.ts`

**Features:**
- Scans 6 knowledge categories (water, fire, shelter, food, navigation, medical)
- 6 quality validation types:
  1. Missing frontmatter (error)
  2. Outdated content >365 days (warning)
  3. Too short <300 words (warning)
  4. Broken links (error)
  5. Missing examples (info)
  6. No tags (info)
- REGEN flagging for critical issues
- Comprehensive HTML report with quality metrics
- Optional batch REGEN script generation

**Interfaces:**
- `QualityIssue`: file, type, severity, message, data
- `GuideMetadata`: path, category, title, frontmatter, content, wordCount
- `QualityReport`: totalGuides, issues, flaggedForRegen, metrics
