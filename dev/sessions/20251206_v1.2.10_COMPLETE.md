## Usage Examples

### Quick Start
```bash
# 1. Start API server
hypercorn extensions.core.webhooks.api_server:app --bind 0.0.0.0:5090

# 2. Start test panel (optional)
python -c "from extensions.core.webhooks.test_panel import create_test_panel; create_test_panel().run()"

# 3. Open browser to http://localhost:5091
```

### Python Client
```python
from extensions.core.webhooks.client import WebhookClient

client = WebhookClient(base_url="http://localhost:5090")

# Add route
client.add_route(
    route_id="github_prs",
    pattern="github.pull_request.*",
    action="log",
    priority=100
)

# Query events
events = client.query_events(platform="github", limit=10)

# Get stats
stats = client.get_stats()
print(f"Total events: {stats.total_events}")
```

### Event Simulation
```python
from extensions.core.webhooks.simulator import WebhookSimulator

simulator = WebhookSimulator()

# Generate event with signature
event = simulator.generate_with_signature('github', 'push',
    repo_name='my-repo',
    commit_message='Fix bug #123'
)

# Send to API
import requests
requests.post(
    'http://localhost:5090/webhooks/github',
    json=event.payload,
    headers=event.headers
)
```

### Platform Integration
```python
from extensions.core.webhooks.integrations import GitHubIntegration

github = GitHubIntegration(token="ghp_...")
webhook = github.create_webhook(
    owner="myorg",
    repo="myrepo",
    url="https://server.com/webhooks/github",
    events=["push", "pull_request"]
)
```

1. **Signature Verification:** All webhook endpoints verify HMAC-SHA256 signatures
2. **Environment Variables:** Secrets stored in `.env` (not committed)
3. **HTTPS Required:** Production webhooks must use HTTPS
4. **Rate Limiting:** Protects against abuse
5. **Input Validation:** All payloads validated before processing
6. **Least Privilege:** API tokens with minimal required permissions
7. **Logging:** All webhook activity logged for audit
8. **Error Handling:** Graceful degradation, no information leakage
