## 🎯 Overview

## 🔧 Technical Implementation

### Architecture

```
External Platform → API Endpoint → Signature Validation → Event Handler → Workflow Trigger
     (GitHub)       /webhooks/      HMAC-SHA256           Platform       uDOS Commands
                                                          Handler
```

### Security Features

1. **HMAC-SHA256 Signature Validation** (all platforms)
2. **Timestamp freshness** (Slack: ±5 min window)
3. **Secret storage** in gitignored `webhooks.json`
4. **IP allowlisting** (optional, documented)
5. **Request logging** (`api_server.log`)

### Event Processing

**GitHub Push Event:**
```python
1. Receive POST /api/webhooks/receive/github
2. Validate X-Hub-Signature-256
3. Parse commits → detect knowledge/ changes
4. Trigger workflows:
   - knowledge-quality-scan (changed guides)
   - xref-validation (all guides)
5. Log event, update trigger count
```

**Slack Slash Command:**
```python
1. Receive POST /api/webhooks/receive/slack
2. Validate X-Slack-Signature + timestamp
3. Parse command: /knowledge water purification
4. Execute: KNOWLEDGE SEARCH water purification
5. Return formatted result to Slack
```

## 📊 Usage Examples

### Register GitHub Webhook

```bash
curl -X POST http://localhost:5000/api/webhooks/register \
  -H "Content-Type: application/json" \
  -d '{
    "platform": "github",
    "events": ["push", "pull_request"],
    "actions": [
      {"event": "push", "workflow": "knowledge-quality-scan"}
    ]
  }'
```

**Response:**
```json
{
  "status": "success",
  "webhook": {
    "id": "wh_a1b2c3d4e5f6g7h8",
    "platform": "github",
    "url": "/api/webhooks/receive/github",
    "secret": "your-secret-here",
    "events": ["push", "pull_request"],
    "created": "2025-12-03T12:00:00"
  }
}
```

### Configure in GitHub

1. Go to repo `Settings` → `Webhooks` → `Add webhook`
2. **Payload URL**: `http://your-server:5000/api/webhooks/receive/github`
3. **Secret**: (use secret from response)
4. **Events**: Pushes, Pull requests, Releases

### Slack Slash Command

```
User: /knowledge water purification
uDOS: [Returns search results from knowledge bank]

User: /map goto AA340
uDOS: [Navigates to Sydney grid cell]

User: /udos FILE LIST
uDOS: [Lists workspace files]
```
