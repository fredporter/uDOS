#### Part 3: Content Gap Analysis (Tasks 6-7)

**Task 6: Gap Detector**
```python
# core/services/knowledge_gap_analyzer.py

class GapAnalyzer:
    def find_gaps(self):
        """Identify missing guides in knowledge matrix."""

        # Expected coverage matrix
        expected = {
            'water': ['purification', 'collection', 'storage', 'testing'],
            'fire': ['starting', 'maintaining', 'safety', 'types'],
            'shelter': ['location', 'materials', 'types', 'insulation'],
            # ...
        }

        # Check what exists
        existing = self.scan_knowledge_bank()

        # Find gaps
        gaps = []
        for category, topics in expected.items():
            for topic in topics:
                if not self.guide_exists(category, topic):
                    gaps.append({
                        'category': category,
                        'topic': topic,
                        'priority': self.calculate_priority(category, topic)
                    })

        return gaps
```

**Task 7: Gap Fill Workflow**
```upy
# memory/workflows/fill-knowledge-gaps.upy

MISSION CREATE "Fill Knowledge Gaps"

# Run gap analysis
WORKFLOW START knowledge-gap-analysis

# Get high-priority gaps
GET $GAPS.HIGH_PRIORITY

# Generate missing guides
FOREACH $gap IN $GAPS.HIGH_PRIORITY
    GENERATE GUIDE $gap.category "$gap.topic"
    CHECKPOINT SAVE "gap-filled-$gap.topic"
END

# Verify coverage
WORKFLOW START knowledge-gap-analysis --verify
```
