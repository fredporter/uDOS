## Three Formats Implemented

### 1. Conditionals

**Short Form (1-2 actions):**
```upy
[IF {$hp} < 30: PRINT ('Low!')]
[IF {$hp} < 30: HP (+20) | PRINT ('Healed!')]
```

**Medium Form (inline branching):**
```upy
[IF {$gold} >= 100 THEN: ITEM (sword) ELSE: PRINT ('Need gold')]
[{$hp} < 30 ? HP (+20) : PRINT ('OK')]  # Ternary
```

**Long Form (complex logic):**
```upy
IF {$hp} < 30
  HP (+50)
  PRINT ('Emergency!')
  FLAG (critical)
ELSE IF {$hp} < 60
  HP (+20)
ELSE
  PRINT ('Healthy')
END IF
```

### 2. Functions

**Short Form (single expression):**
```upy
@add({$a}|{$b}): RETURN {$a} + {$b}
@greet({$name}): PRINT ('Hello {$name}!')
```

**Medium Form (multi-action):**
```upy
@heal({$amt}): SET {$SPRITE-HP|{$SPRITE-HP} + {$amt}} | PRINT ('Healed!')
```

**Long Form (complex logic):**
```upy
FUNCTION battle_check({$level})
  ROLL (1d20) → {$roll}
  IF {$roll} >= 15
    XP (+50)
    RETURN true
  ELSE
    RETURN false
  END IF
END FUNCTION
```

## Enhanced Delimiter System

| Symbol | Purpose | Example |
|:-------|:--------|:--------|
| `:` | THEN separator | `[IF {$hp} < 30: PRINT ('Low')]` |
| `\|` | Multiple actions | `HP (+20) \| PRINT ('Healed!')` |
| `\|` | Function params | `@func({$a}\|{$b})` |
| `?` | Ternary condition | `[{$hp} < 30 ? HP (+20) : PRINT ('OK')]` |
| `→` | Result assignment | `ROLL (1d20) → {$result}` |
| `THEN` | Explicit branch | `[IF ... THEN: ... ELSE: ...]` |
| `ELSE` | Alternative | `ELSE IF` / `ELSE` |
| `END IF` | Close conditional | Long form only |
| `END FUNCTION` | Close function | Long form only |

## Key Benefits

### 1. Progressive Complexity

- **Beginners**: Start with short form only
- **Intermediate**: Add medium form for branching
- **Advanced**: Long form for complex logic

### 2. No Indentation Required

All formats work line-by-line:
- No tab vs space confusion
- Easier parsing
- Copy-paste friendly

### 3. Human-Readable

Explicit keywords aid understanding:
- `THEN` shows consequence
- `ELSE` shows alternative
- `END IF` closes blocks
- `END FUNCTION` closes functions

### 4. Visual Distinction

Three bracket types = three purposes:
- `{$variable}` = data containers
- `(command)` = actions/functions
- `[condition]` = control flow

### 5. Flexibility

Choose the format that fits:
- Quick check? Short form
- Binary choice? Medium form with THEN/ELSE
- Complex logic? Long form with multiple branches
