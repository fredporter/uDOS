### 🎯 API Endpoints

**Webhook Receivers:**
- `POST /webhooks/github` - Receive GitHub events
- `POST /webhooks/slack` - Receive Slack events
- `POST /webhooks/notion` - Receive Notion events
- `POST /webhooks/clickup` - Receive ClickUp events

**Event Management:**
- `GET /api/events` - Query event history (filter by platform, type, limit)
- `GET /api/events/{id}` - Get specific event

**Route Management:**
- `GET /api/routes` - List all routes
- `GET /api/routes/{name}` - Get specific route
- `POST /api/routes/{name}/enable` - Enable route
- `POST /api/routes/{name}/disable` - Disable route
- `DELETE /api/routes/{name}` - Remove route

**System:**
- `GET /api/stats` - Get statistics (API, processor, router)
- `POST /api/stats/reset` - Reset all stats
- `GET /api/health` - Health check

### 🔒 Security Features

- **Webhook Signature Verification:**
  - GitHub: HMAC SHA-256 (X-Hub-Signature-256)
  - Slack: HMAC SHA-256 with timestamp (X-Slack-Signature)
  - Notion: HMAC SHA-256 (Notion-Signature)
  - ClickUp: HMAC SHA-256 (ClickUp-Signature)

- **API Authentication:**
  - Bearer token support
  - Configurable API keys
  - Rate limiting (100 req/min default)

- **CORS:**
  - Configurable origins
  - Cross-origin support for web clients

### 📚 Client Usage Examples

**Synchronous:**
```python
from extensions.core.webhooks.client import WebhookClient

client = WebhookClient('http://localhost:5050')

# Get recent events
events = client.get_events(platform='github', limit=10)

# Get route info
routes = client.get_routes()
route = client.get_route('notify_push')

# Manage routes
client.enable_route('notify_push')
client.disable_route('notify_push')

# Get statistics
stats = client.get_stats()
print(f"Processed: {stats.processor['processed']}")

# Health check
health = client.health_check()
```

**Asynchronous:**
```python
async with WebhookClient('http://localhost:5050') as client:
    events = await client.get_events_async(platform='slack')
    routes = await client.get_routes_async()
    stats = await client.get_stats_async()
```
