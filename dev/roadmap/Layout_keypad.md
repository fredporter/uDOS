Layout (keypad style)

+------+------+------+
|  7   |  8   |  9   |
| Redo |  ↑   | Undo |
+------+------+------+
|  4   |  5   |  6   |
|  ←   |  OK  |  →   |
+------+------+------+
|  1   |  2   |  3   |
|  Yes |  ↓   |  No  |
+------+------+------+
       |  0   |
       | Menu |
       +------+

What each key does
	•	Movement (4-way arrows)
	•	8 → Move Up
	•	2 → Move Down
	•	4 → Move Left
	•	6 → Move Right
	•	Editing
	•	7 → Redo 
	•	9 → Undo 
	•	5 → Select / Confirm / Enter / OK(centre “action” button)
	•	Question navigation
	•	1 → Yes 
	•	3 → No
	•	0 → Menu /
