- **Rate limit:** 2 req/sec (prevents accidental spam)

- **Alerts:** 80% budget warning, 100% hard stop

## 🔄 Migration Path

### Command Changes
```upy
# OLD (deprecated)
ASSISTANT ASK how do I purify water?
OK ASK what's the best fire method?
OK CLEAR
OK STATUS

# NEW (recommended)
GENERATE DO how do I purify water?
GENERATE DO what's the best fire method?
GENERATE CLEAR
GENERATE STATUS
```

## 🎨 Usage Examples

### Basic Query (Offline)
```upy
GENERATE DO how do I purify water?

# Output:
✅ Knowledge Bank (Offline - No API Cost)
Confidence: 85%

[Answer with sources]
```

### Retry with Modifications
```upy
GENERATE DO fire starting methods
GENERATE REDO without matches
GENERATE REDO in wet conditions
```

### Workflow Control
```upy
# Force offline mode
SET GENERATE.MODE offline
GENERATE DO water sources

# Monitor budget
GET API.BUDGET_REMAINING
# Output: 0.9988

# Conditional generation
IF $API.BUDGET_PERCENT < 80 THEN
  GENERATE DO complex query
ELSE
  GENERATE DO --mode offline simple query
ENDIF
```

### Custom Prompts
```upy
SET PROMPT.SYSTEM "Expert wilderness survival instructor"
SET PROMPT.TONE professional
SET PROMPT.COMPLEXITY detailed

GENERATE DO shelter building basics
```
