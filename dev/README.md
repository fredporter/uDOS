# uDOS Development Environment

This directory contains development tools and templates for uDOS core system development.

## Structure (Local Only - Generated on Install)

```
dev/
├── active/           # Current development work (LOCAL ONLY)
├── scripts/          # Development automation (LOCAL ONLY)
├── docs/             # Architecture documentation (LOCAL ONLY)
├── copilot/          # AI assistant context (LOCAL ONLY)
├── vscode/           # VS Code configurations (LOCAL ONLY)
├── roadmaps/         # Development roadmaps (LOCAL ONLY)
└── templates/        # Development templates (SYNCED)
```

## Access Requirements

- **Role**: WIZARD (Level 100) required
- **Mode**: DEV mode must be activated
- **Purpose**: Core system development only

## Installation

The development environment is automatically set up during installation. Most contents are generated locally and not synced to GitHub to keep the repository minimal.

Use `./dev/docs/SETUP-COMPLETE.md` for complete development setup instructions.
