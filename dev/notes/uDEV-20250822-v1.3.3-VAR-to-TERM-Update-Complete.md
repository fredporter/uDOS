### 2. **Syntax Change: VAR → TERM**

**Before:**
```markdown
[VAR] {USERNAME}
[VAR] {USERNAME | <SLUGIFY>}
[VAR] {CREATED-DATE | <FORMAT-TIMESTAMP>}
```

**After:**
```markdown
[TERM] {USERNAME}
[TERM] {USERNAME | <SLUGIFY>}
[TERM] {CREATED-DATE | <FORMAT-TIMESTAMP>}
```

### 📋 **Current uCODE Block Syntax**

| **Element** | **Syntax** | **Example** |
|-------------|------------|-------------|
| Variable | `[TERM] {NAME}` | `[TERM] {USERNAME}` |
| Variable + Function | `[TERM] {NAME \| <FUNC>}` | `[TERM] {USERNAME \| <SLUGIFY>}` |
| Command | `[COMMAND] {PARAMS}` | `[GET-RETRIEVE] {USER-DATA \| USERNAME}` |
| Conditional | `[IF] {CONDITION}...[/IF]` | `[IF] {FULL-NAME}...[/IF]` |
| Loop | `[EACH] {COLLECTION}...[/EACH]` | `[EACH] {PREFERENCES}...[/EACH]` |
| Context | `[WITH] DATA...[/WITH]` | `[WITH] [GET-RETRIEVE] {...}...[/WITH]` |
| Function | `<FUNCTION-NAME>` | `<FORMAT-TIMESTAMP>` |
