  - `PREFERENCE-UPDATE` - User preference changes
  - `LOCATION-SET` - Location updates
  - `FORM-SUBMISSION` - Form data processing

- **Location:** `/uMEMORY/system/`


#### **5. Updated Template Format**
- **Location:** `/uMEMORY/templates/input-user-setup.md`
- **New Syntax:**
  - Variables: `{{USERNAME}}` (UPPERCASE-DASH)
  - Commands: `{{[GET-RETRIEVE] {SETUP-STATUS | USERNAME}}}`
  - Functions: `{{USERNAME | <SLUGIFY>}}`
  - VB-Style: `{{#IF CONDITION}}...{{/IF}}`

### 🎯 New uCODE Standards

#### **Command Format**
```bash
[COMMAND-NAME] {PARAMETER | OPTIONS}
```

#### **Function Format**
```bash
<FUNCTION-NAME>
```

#### **Variable Format**
```bash
{VARIABLE-NAME | PIPE-OPTIONS}
```

#### **Template Syntax**
```handlebars
{{VARIABLE | <FUNCTION>}}
{{[COMMAND] {PARAMS}}}
{{#IF CONDITION}}...{{ELSE}}...{{/IF}}
{{#EACH COLLECTION}}...{{/EACH}}
{{#WITH OBJECT}}...{{/WITH}}
```
