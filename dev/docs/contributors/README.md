# Contributor Shared Area

Updated: 2026-03-05

This folder is the shared contributor handoff lane under `@dev`.

Use this area for:
- contributor submissions that need review
- contributor-side roadmap proposals
- local devlog fragments that are not yet promoted into canonical contributor docs
- contributor decision drafts that are not yet promoted

Promote mature content into canonical `dev/docs/decisions/`, `dev/docs/devlog/`, and `dev/docs/roadmap/`.
