# Contributor Task Management

Updated: 2026-03-04

Task management for the `@dev` workspace is split across:

- `dev/ops/tasks.md`: active work and blockers
- `dev/ops/tasks.json`: machine-readable contributor task state
- `dev/ops/completed.json`: completed milestones
- `dev/ops/DEVLOG.md`: narrative execution log

Use this directory for task-management conventions and indexes. Do not place runtime operator task lists here; those belong with their owning runtime subsystem.
