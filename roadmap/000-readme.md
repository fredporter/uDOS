# 🗺️ uOS Roadmap Index

Welcome to the `roadmap/` directory of the uOS project — a chronologically ordered library of system design, philosophy, structure, and world-building.

Each file is numbered according to the **AI Move** that created or refined it. This ensures full traceability of uOS evolution, one Move at a time.

---

## 📂 Folder Structure

| Folder          | Description |
|------------------|-------------|
| `01-core/`       | Foundational values, terminology, and high-level orientation of uOS philosophy. |
| `02-structure/`  | Templates and patterns for Moves, Milestones, Missions, and Memory. |
| `03-ux/`         | UI concepts, dashboard, CLI interaction philosophy, ASCII design patterns. |
| `04-knowledge/`  | Map logic, Tower of Knowledge, spatial navigation, and public resource structure. |
| `05-system/`     | uScript format, backend logic, data flow, and system features. |
| `legacy/`        | Preserved early spells and concept sketches before numbering. |

---

## 🔢 Filename Convention

```
<3-digit AI Move Number>-<kebab-case-title>.md
```

Examples:

- `018-wizard-tower-map.md`
- `023-move-log-dashboard.md`
- `041-uknowledge-map-logic.md`

Avoid symbols, capitalisation, or underscores.
