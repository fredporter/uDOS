# uOS Documentation

This archive contains the core documentation files for uOS, a private AI-based operating system designed as a lifelong personal assistant and knowledge library.

## Contents

- `uos_overview.md`: Overview of uOS, core principles, terminology, and development approach.
- `uos_terminology.md`: Key terminology and naming conventions used within uOS.
- `uos_development_guidelines.md`: Development guidelines, principles, and workflow recommendations.

## About uOS

uOS operates on a privacy-first, local-only model, with a unique device/NFT binding. Interaction and data are fully Markdown-based. The system uses atomic input/output operations called **Moves**, which aggregate into **Milestones**, contributing to user **Missions** and an ultimate **Legacy**.

## Usage

- Keep all files updated and version controlled.
- Use these docs as a reference when developing or interacting with uOS.
- All user interaction and development output should be recorded in Markdown for consistency and auditability.
