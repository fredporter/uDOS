This document outlines future improvements, experimental features, and roadmap items that have not yet been fully implemented in uDOS. It acts as a rolling concept log and planning area for new mechanics, interface layers, and philosophy extensions.

## 🌀 Execution Loop Enhancements

- Add support for pre- and post-move hooks
- Interactive logging preview (before committing)
- Loop-level step diagnostics

---

## 🌍 Tour Mode

- Offline-only personal assistant with reduced memory footprint
- Local prompts and maps bundled into single-use image
- Temporary memory map without persistent state

---

## 🤖 AI and Agent Expansion

- User-defined agents tied to Missions
- Long-term memory anchor via sandbox
- Scriptable personalities (non-Otter)
- Custom dashboards per agent

---

## 🧱 Dashboard Evolution

- Expand dashboard templates with conditional logic
- Animated updates (ASCII render diffs)
- Mini-maps linked to uMap tile or active region
- Markdown chart support for logs and metrics

---

## 🔐 System Identity and Security

- Optional physical signature binding
- Immutable instance identity (hash of install + boot + device)
- Companion user profile logic
- No usernames/logins stored outside `sandbox/user.md`

---

## 🛠️ Devtools and Debugging

- Inline step tracker for scripts
- Real-time Markdown preview of upcoming move
- Log file versioning and patching
