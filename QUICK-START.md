# Quick Start

Use this page as the thin runtime entry. Full setup details live in `docs/`.

## One Entrypoint

Install or update through the canonical launcher:

```bash
./bin/udos install
```

Common install modes:

```bash
./bin/udos install --core
./bin/udos install --wizard
./bin/udos install --update
```

## Launch

Terminal:

```bash
./bin/udos
```

macOS Finder:

- Double-click `bin/ucode-tui-v1.5.command`

## First Commands

```bash
ucode HELP
ucode SETUP
ucode STATUS
ucode UCODE PROFILE LIST
ucode UCODE OPERATOR STATUS
```
