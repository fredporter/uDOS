## Processing Instructions

### 1. Load Template
```ucode
LOAD TEMPLATE {{TEMPLATE_NAME}}
VALIDATE TEMPLATE STRUCTURE
LOG "Template loaded: {{TEMPLATE_NAME}}"
```

### 2. Load Data
```ucode
LOAD DATA {{DATA_SOURCE}}
VALIDATE DATA FORMAT
SET DATA_COUNT = COUNT(DATA_ROWS)
LOG "Data loaded: {{DATA_COUNT}} records"
```

### 3. Process Variables
```ucode
FOR EACH VARIABLE IN TEMPLATE:
    IF VARIABLE_TYPE = "SYSTEM":
        SET VARIABLE_VALUE = GET_SYSTEM_VALUE(VARIABLE_NAME)
    ELIF VARIABLE_TYPE = "DATA":
        SET VARIABLE_VALUE = GET_DATA_VALUE(VARIABLE_NAME)
    ELIF VARIABLE_TYPE = "USER":
        SET VARIABLE_VALUE = PROMPT_USER(VARIABLE_NAME)
    ENDIF
    REPLACE TEMPLATE_VARIABLE WITH VARIABLE_VALUE
NEXT VARIABLE
```

### 4. Generate Output
```ucode
PROCESS TEMPLATE WITH DATA
GENERATE MARKDOWN OUTPUT
VALIDATE OUTPUT FORMAT
SAVE TO {{OUTPUT_FILE}}
LOG "Report generated: {{OUTPUT_FILE}}"
```
