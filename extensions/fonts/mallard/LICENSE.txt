## Variants Description

- **blocky**: Base version, standard Teletext rendering
- **blockier**: More angular, sharper edges
- **smooth**: Softened edges for better screen rendering
- **smoother**: Extra smoothing for modern displays
- **neueue**: Modern interpretation with updated metrics
- **tiny**: Compact variant for dense layouts
