üé® CSS Framework Details

system.css recreates the classic Macintosh System OS interface that ran from 1984 to 1991.

This library is based on System 6, the final monochrome version of macOS, providing:

  • Window frames with title bars and resize handles
  • Classic button and form element styling
  • Authentic scrollbars and dialog boxes
  • Proper spacing and typography

Installation:

<link rel="stylesheet" href="https://unpkg.com/@sakun/system.css" />

Basic Usage:

<div class="window">
  <div class="title-bar">
    <button class="close"></button>
    <h1 class="title">Window Title</h1>
    <button class="resize"></button>
  </div>

  </div>
</div>

About system.css

System.css is a CSS library for building interfaces that resemble Apple's System OS which ran from 1984-1991. This library is based on System 6 as it was the final monochrome version of macOS.

The framework provides authentic classic Mac interface components including windows, buttons, forms, and typography that captures the essence of early Macintosh computing.

Key Features:

  • Authentic System 6 monochrome interface
  • No JavaScript dependencies
  • Compatible with modern front-end frameworks
  • Customizable styles and overrides
  • Responsive design support
