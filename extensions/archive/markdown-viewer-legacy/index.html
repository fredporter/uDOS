Features

  • GitHub-Flavored Markdown - Full GFM support with tables, task lists, and more
  • Light/Dark Themes - Toggle between GitHub light and dark styles
  • uCODE Support - Special rendering for uDOS command syntax
  • PANEL Support - Interactive info panels and callouts
  • Syntax Highlighting - Code blocks with language detection
  • Fuzzy Search - Typo-tolerant file search
  • Split View - View source and rendered side-by-side

Keyboard Shortcuts

Key Action
Ctrl+/Toggle Light/Dark
Ctrl+UToggle uCODE Panel
Ctrl+PToggle Info Panel
Ctrl+FFocus Search
Ctrl+SSave/Download

uCODE Syntax

Use special [COMMAND|ARG|VALUE] syntax for interactive commands:

[FILE|READ|/path/to/file.txt]
[SYSTEM|INFO]
[HELP|COMMAND|grep]
[WEB|START|dashboard]

PANEL Syntax

Create callout panels with:

:::note
This is an informational panel
:::

:::warning
This is a warning panel
:::

:::danger
This is a danger/error panel
:::

:::success
This is a success panel
:::
