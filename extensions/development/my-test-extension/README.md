## Integration with uDOS

This extension can be managed using uDOS POKE commands:

```bash
# Start the extension
POKE START my-test-extension

# Check status
POKE STATUS my-test-extension

# Get information
POKE INFO my-test-extension

# Stop the extension
POKE STOP my-test-extension
```
