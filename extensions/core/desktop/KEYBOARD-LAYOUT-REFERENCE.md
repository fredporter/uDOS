## Tips & Tricks

1. **Mix Character Sets**: Switch between sets to combine styles
2. **Use Copy/Paste**: Duplicate complex patterns quickly
3. **Start with Templates**: Begin with existing characters, modify
4. **Preview Often**: Check rendering in preview pane
5. **Save Frequently**: Export JSON backups regularly

## Unicode Reference

### Block Elements (U+2580вҖ“U+259F)
- Upper half: в–Җ (U+2580)
- Lower half: в–„ (U+2584)
- Full block: в–Ҳ (U+2588)
- Left half: в–Ң (U+258C)
- Right half: в–җ (U+2590)

### Box Drawing (U+2500вҖ“U+257F)
- Horizontal: в”Җ (U+2500)
- Vertical: в”Ӯ (U+2502)
- Corners: в”Ң в”җ в”” в”ҳ
- Intersections: в”ң в”Ө в”¬ в”ҙ в”ј

### Geometric Shapes (U+25A0вҖ“U+25FF)
- Black square: в–  (U+25A0)
- White square: в–Ў (U+25A1)
- Black diamond: в—Ҷ (U+25C6)
- White diamond: в—Ү (U+25C7)
- Black circle: в—Ҹ (U+25CF)
- White circle: в—Ӣ (U+25CB)
