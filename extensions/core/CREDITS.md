## 📺 Teletext Extension

### Standards
- **BBC Teletext Level 1 (1976)**
  - 40×25 character grid specification
  - Mode 7 graphics standard
  - Control codes and page system

### Fonts
- **Mallard Font Family**
  - 6 variants for authentic BBC Teletext rendering
  - Author: "gid" (FontStruct.com)
  - License: Creative Commons Attribution Share Alike 3.0
  - Variants: Blocky, Blockier, Smooth, Smoother, Neueue, Tiny
  - Location: `/extensions/assets/fonts/mallard/`

### Historical Reference
- **BBC CEEFAX (1974-2012)**
  - First operational teletext service
  - Page numbering system (100-899)

### Color Enhancement
