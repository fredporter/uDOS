# system.css Credits & Attribution

## About system.css

**system.css** is a CSS library for building interfaces that resemble Apple's System OS which ran from 1984-1991. This library is based on the final monochrome version of macOS, System 6.

- **Project**: system.css
- **Version**: v0.1.11
- **Author**: Sakun Acharige (@sakofchit)

## Font Credits

The authentic Mac OS fonts included in this distribution were meticulously recreated by **@blogmywiki**:

### Chicago_12 (ChiKareGo2)
- Classic Mac system font
- Used throughout Mac OS for menus, dialogs, and UI elements
- Recreated by @blogmywiki

### Geneva_9 (FindersKeepers)
- Small system font used for labels and fine print
- Used in Finder and system dialogs
- Recreated by @blogmywiki

### ChicagoFLF
- Display version of Chicago
- Used for headings and titles
- Recreated by @blogmywiki

### Monaco
- Classic Mac monospace font
- Used for code, terminal, and technical displays
- Recreated by @blogmywiki

## Integration in uDOS

This distribution of system.css assets is used in the uDOS Desktop environment to provide an authentic classic Mac OS experience. The fonts and icons are used as follows:

### Fonts
- **Primary UI**: Chicago_12 for all user interface elements
- **Headers**: Chicago for headings and titles
- **Small Text**: Geneva_9 for labels and captions
- **Monospace**: Monaco for terminal and code displays

### Icons
- **Form Controls**: Authentic system.css icons for checkboxes, radio buttons
- **Buttons**: system.css button borders for authentic look
- **Scrollbars**: system.css scrollbar arrows and controls
- **Desktop Icons**: Supplemented with Mono Icon Theme (see separate credits)
