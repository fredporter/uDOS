# Extensions Documentation

**Public-facing uDOS extensions**

## Core Extensions (Public)

These extensions are included in the public uDOS distribution:

- **[api/](../api/)** - REST/WebSocket API Server
- **[transport/](../transport/)** - Private transport layer (QR, Audio, MeshCore, Bluetooth)
- **[vscode/](../vscode/)** - VS Code extension for uDOS development

## Wizard-Only Extensions

AI and experimental features are in `/wizard/extensions/`:

- **assistant/** - AI services (Gemini, Vibe CLI, Ollama)
- **groovebox/** - Music production engine (MML, 808 drums)
- **tauri/** - Legacy Tauri integration
- **assets/** - Shared wizard assets

See [/wizard/extensions/](../../wizard/extensions/) for wizard-only features.
