# uDOS Sandbox - User Workspace

This directory is your active workspace for experimentation, logging, and temporary files.

## Structure (Generated on First Use)

```
sandbox/
├── logs/             # System and session logs
├── backup/           # User backups and archives
├── sessions/         # Active session data
├── scripts/          # User scripts and experiments
├── tasks/            # Task management files
├── experiments/      # Development experiments
├── generated/        # Auto-generated content
├── user/             # User-specific data
└── workflow/         # Workflow and mission data
```

## Purpose

- **Active workspace** for all user development and experimentation
- **Flushable environment** - contents can be reset/cleared
- **Log storage** for system operations and user activities
- **Temporary storage** for work-in-progress files

## Data Management

- All contents are **LOCAL ONLY** (not synced to GitHub)
- Important work should be moved to `uMEMORY/` for permanent storage
- Regular cleanup and archiving recommended
- Automatic backup system available

## Access

- Available to all user roles (GHOST through WIZARD)
- No special permissions required
- Safe environment for experimentation
