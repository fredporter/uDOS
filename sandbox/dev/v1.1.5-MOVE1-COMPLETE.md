## Technical Implementation

### SVG Styles

| Style      | Description                           | Colors                  | Use Case              |
|------------|---------------------------------------|-------------------------|-----------------------|
| lineart    | Clean black lines, minimal            | #000 (black)            | Simple diagrams       |
| blueprint  | Technical drawing, blue background    | #003366 bg, #4A90E2 fg  | Technical specs       |
| sketch     | Hand-drawn, organic feel              | Grays (#333, #666)      | Informal illustrations|
| isometric  | 3D projection, 30° angles             | Multiple (RGB)          | 3D views              |

### AI Integration

- **Provider**: Google Gemini (1.5 Flash, 1.5 Pro)
- **Fallback**: Template generator (offline mode)
- **Prompts**: Style-specific, detailed specifications
- **Validation**: XML parsing (well-formedness)
- **Post-Processing**:
  - Add xmlns if missing
  - Add viewBox if missing
  - Remove namespace prefixes (ns0:svg → svg)
  - Pretty print (indent)

### Command Syntax

```bash
# Basic usage
SVG <description>

# With style
SVG <description> --style <lineart|blueprint|sketch|isometric>

# With save
SVG <description> --save <filename>

# All options
SVG <description> --style <style> --save <file> --no-preview
```

### Examples

```bash
SVG water filter diagram
SVG fire triangle --style blueprint
SVG shelter types --style isometric --save shelter.svg
SVG first aid steps --style sketch --no-preview
```

## Architecture Notes

### Extension Design
- **Optional**: Not loaded by default (performance)
- **Lazy Loading**: Imports on first SVG command
- **Graceful Degradation**: Works without Gemini API (templates)
- **Self-Contained**: No core system dependencies

### Integration Points
- **Command System**: Clean routing through uDOS_commands.py
- **Config System**: Uses core.config.Config
- **Logging**: Integrates with core logger
- **File I/O**: Uses sandbox/drafts/svg/ output

### Design Decisions
1. **Module Naming**: `svg_generator` not `svg-generator` (Python compatibility)
2. **Command Name**: `SVG` (concise, clear, not conflicting with GENERATE)
3. **Fallback Templates**: Ensure offline functionality
4. **Namespace Cleanup**: Remove ns0: prefixes for cleaner SVG
5. **Auto-save**: Default to sandbox/drafts/svg/ with timestamp
