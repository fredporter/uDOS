## Technical Architecture

### Flat Directory Structure

```
sandbox/logs/
├── session-commands-2025-11-28.log      # Session logs (INPUT/OUTPUT/MOVE)
├── system-startup-2025-11-28.log        # System startup logs
├── ucode-execution-2025-11-28.log       # uCODE interpreter logs
├── command-ask-2025-11-28.log           # ASK command logs
├── web-api-request-2025-11-28.log       # Web API request logs
├── ai-gemini-request-2025-11-28.log     # AI assistant logs
├── extension-loader-2025-11-28.log      # Extension system logs
├── error-critical-2025-11-28.log        # Critical errors (90 days)
├── debug-parser-2025-11-28.log          # Debug logs (7 days)
├── audit-file-access-2025-11-28.log     # Audit logs (90 days)
└── mission-complete-2025-11-28.log      # Mission logs (never deleted)
```

### Log Format Examples

**Standard Format**:
```
[2025-11-28 10:30:45] [INFO] [session-commands] SESSION START
[2025-11-28 10:30:50] [INFO] [session-commands] [INPUT] ASK what is water
[2025-11-28 10:30:52] [INFO] [session-commands] [OUTPUT] Water is H2O...
[2025-11-28 10:30:52] [INFO] [session-commands] [MOVE] 1 (Total: 42)
```

**Structured Format (JSON)**:
```json
{"timestamp": "2025-11-28T10:30:45", "level": "INFO", "category": "api-request", "message": "GET /status", "context": {"user": "fred", "session": "abc123", "ip": "127.0.0.1"}}
```

### Retention Policies

| Category       | Retention | Auto-Delete |
|----------------|-----------|-------------|

## Usage Examples

### Basic Logging

```python
from core.services.logging_manager import get_logger

logger = get_logger('system-startup')
logger.info('uDOS starting...')
logger.debug('Loading configuration...')
logger.error('Failed to load file')
```

### Structured Logging

```python
from core.services.logging_manager import get_logger

logger = get_logger('api-requests',
                   structured=True,
                   context={'user': 'fred', 'session': 'abc123'})
logger.info('API request received')
```

### Session Logging (Backward Compatible)

```python
from core.services.session_logger import SessionLogger

logger = SessionLogger()
logger.log('INPUT', 'ASK what is water')
logger.log('OUTPUT', 'Water is H2O...')
# Automatically tracks move: INPUT + OUTPUT = 1 move
```

### Search Logs

```python
from core.services.logging_manager import get_logging_manager

mgr = get_logging_manager()
results = mgr.search_logs('error', category='web-api', days=7)
for filepath, lineno, content in results:
    print(f'{filepath}:{lineno} - {content}')
```

### Log Statistics

```python
from core.services.logging_manager import get_logging_manager

mgr = get_logging_manager()
stats = mgr.get_log_stats()
print(f"Total files: {stats['total_files']}")
print(f"Total size: {stats['total_size_mb']:.2f} MB")
```

### Retention Enforcement

```python
from core.services.logging_manager import get_logging_manager

mgr = get_logging_manager()

# Dry run (see what would be deleted)
deleted = mgr.enforce_retention_policy(dry_run=True)

# Actually delete
deleted = mgr.enforce_retention_policy(dry_run=False)
```
