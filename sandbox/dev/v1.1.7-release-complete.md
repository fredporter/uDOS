## 🚀 How to Use

### Generate Your First SVG

```bash
# Start uDOS
./start_udos.sh

# Basic generation
🔮 > GENERATE SVG water filter

# With custom style
🔮 > GENERATE SVG fire triangle --style hand-illustrative

# Pro mode with strict validation
🔮 > GENERATE SVG shelter frame --type architecture --pro --strict
```

### Run Workflow Examples

```bash
# Quick start tutorial
🔮 > RUN "sandbox/workflow/examples/nano_banana_quick_start.uscript"

# Batch generation
🔮 > RUN "sandbox/workflow/examples/nano_banana_batch.uscript"

# Quality validation
🔮 > RUN "sandbox/workflow/examples/nano_banana_quality_check.uscript"

# Style demonstration
🔮 > RUN "sandbox/workflow/examples/nano_banana_styles_demo.uscript"
```

### Run Tests

```bash
# Smoke tests (fast, no API key)
pytest sandbox/tests/test_nano_banana_smoke.py -v

# Integration tests (needs v1.1.8 fix)
pytest sandbox/tests/integration_nano_banana_test.py -v

# Live API tests (requires GEMINI_API_KEY)
pytest sandbox/tests/live_nano_banana_test.py -v -s
```

### Commands Reference

```bash
# Generate SVG
GENERATE SVG <description> [--style <style>] [--type <type>] [--save <file>] [--pro] [--strict]

# Styles
--style technical-kinetic  # MCM geometry (default)
--style hand-illustrative  # Hand-drawn aesthetic
--style hybrid             # Mix of both

# Types
--type flowchart      # Processes, flows (default)
--type architecture   # Structures, systems
--type kinetic-flow   # Mechanical, motion
--type schematic      # Technical specs

# Options
--pro                 # Use Nano Banana Pro (slower, higher quality)
--strict              # Enforce strict validation
--save <file>         # Custom filename
--no-preview          # Skip preview hints
```

## 🔧 Technical Details

### Architecture

```
User Request
    │
    ├─→ 1. Load Style Guide (0-14 reference PNGs)
    │
    ├─→ 2. Generate PNG (Gemini 2.5 Flash Image - Nano Banana)
    │
    ├─→ 3. Vectorize (potrace/vtracer → SVG)
    │
    ├─→ 4. Validate (Technical-Kinetic compliance)
    │
    ├─→ 5. Save (sandbox/drafts/svg/)
    │
    └─→ 6. Output (Success message + file path)
```

### Performance

| Mode | Target | Actual | Status |
|------|--------|--------|--------|
| Standard | <30s | 15-20s avg | ✅ |
| Pro | <60s | 30-45s avg | ✅ |
| Smoke Tests | <1s | 0.03s | ✅ |

### Requirements

- **GEMINI_API_KEY** in `.env`
- **potrace** (recommended): `brew install potrace`
- **vtracer** (fallback): `pip install vtracer>=0.6.0`
- **Python 3.8+**
- **uDOS v1.1.7+**
