## Session Overview

First implementation session for **Nano Banana SVG Generation** system - a complete redesign of uDOS's SVG generation using Gemini 2.5 Flash Image (Nano Banana) with PNG-to-SVG vectorization pipeline.### Core Concept

Replace unreliable text-to-SVG generation with robust **PNG → SVG pipeline**:

1. Load style guide (up to 14 reference PNGs)
2. Generate high-quality monochrome PNG with Nano Banana
3. Vectorize PNG to clean SVG (potrace/vtracer)
4. Validate Technical-Kinetic compliance
5. Save editable SVG

### 3. ✅ Vectorization Service

**File:** `core/services/vectorizer.py` (360 lines, NEW)

**Features:**
- Potrace integration (primary method)
- vtracer fallback (Rust-based)
- Technical-Kinetic validation
- Stroke width normalization (2.5px default)
- XML well-formedness checking
- Monochrome enforcement (black/white only)

**API Usage:**
```python
from core.services.vectorizer import get_vectorizer_service

vectorizer = get_vectorizer_service()

# Vectorize PNG → SVG
result = vectorizer.vectorize(
    png_bytes,
    stroke_width=2.5,
    simplify=True,
    validate_compliance=True
)

# Check results
if result.validation["compliant"]:
    svg_content = result.svg_content
    print(f"Vectorized using {result.method}")
else:
    print("Validation errors:", result.validation["errors"])
```

**Validation Checks:**
- ✅ Valid XML (well-formed SVG)
- ✅ No forbidden elements (gradient, filter, pattern, image)
- ✅ Monochrome only (black #000 / white #FFF)

## Technical Notes

### Gemini Model Names

**Current (as of Nov 2025):**
- Text: `gemini-2.5-flash`
- Image: `gemini-2.0-flash-exp` (Nano Banana)
- Image Pro: `gemini-exp-1206` (Nano Banana Pro)

**Note:** Model names may change. Check Google AI Studio for latest.

### Vectorization Quality

**Potrace advantages:**
- Industry standard (used by Inkscape)
- Highest quality output
- Excellent corner detection
- Clean path simplification

**vtracer advantages:**
- Python library (easier install)
- Rust-based (fast)
- Good for fallback

**Recommendation:** Install potrace if possible, use vtracer as fallback.

### Style Guide Best Practices

**Reference Image Requirements:**
- **Format:** PNG (high contrast)
- **Resolution:** 1200x900 pixels minimum
- **DPI:** 300
- **Colors:** ONLY black (#000000) and white (#FFFFFF)
- **Stroke:** 2-3px consistent
- **Count:** 7-14 images per style

**Demonstration Focus:**
- 1 image per key concept
- Clear, unambiguous examples
- High contrast, sharp edges
- No text (unless demonstrating typography)
