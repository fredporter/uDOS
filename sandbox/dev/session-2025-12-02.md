## Technical Details

### Service APIs Used

**XPService:**
- `get_total_xp()` → int: Returns sum of XP across all categories
- Level calculation: `while level ** 2 * 100 <= xp: level += 1`
- Current level XP: `(level - 1) ** 2 * 100`
- Next level XP: `level ** 2 * 100`

**InventoryService:**
- `get_inventory(location="Personal Inventory")` → List[Dict]
  - Returns list of items with name, quantity, condition, etc.
- `get_inventory_stats(location)` → Dict
  - Returns total_weight, item_count (unique items)

**SurvivalService (from previous work):**
- `get_stat(SurvivalStat)` → Dict with current/max/min/status
- Status calculation with thresholds for critical/warning/normal

### Progress Bar Implementation

20-character Unicode progress bar using block characters:
```python
progress = int((xp_in_level / xp_needed) * 20)
bar = "█" * progress + "░" * (20 - progress)
```

Examples:
- 0%: `[░░░░░░░░░░░░░░░░░░░░] 0%`
