## 🎮 Command Usage

### CLEAN Command

```bash
# Interactive menu
CLEAN

# Clean specific folder
CLEAN logs

# Clean multiple folders
CLEAN logs,trash,drafts

# Clean all (no prompts)
CLEAN --all

# Preview deletions
CLEAN --dry-run

# Show statistics only
CLEAN --stats

# Keep last 7 days
CLEAN logs --days=7

# Reset to pristine state
CLEAN --reset

# Override protection
CLEAN tests --force
```

### TIDY Command

```bash
# Organize all folders
TIDY

# Organize specific folder
TIDY logs

# Organize multiple folders
TIDY logs,scripts

# Report without changes
TIDY --report

# Include statistics
TIDY --stats

# Auto-organize (no prompts)
TIDY --auto
```

```bash
# REPAIR with sandbox health check

# DESTROY with sandbox cleanup
DESTROY --reset    # Safe reset (preserves user/tests)

DESTROY --all      # Nuclear option (DANGER)
```

### DESTROY Command

```python
# In system_handler.py
def handle_destroy(self, params, grid, parser):
    """Destructive reset with sandbox integration."""
    destruction_type = params[0]

    mode_map = {
        "--reset": "reset",  # Safe (preserves user/tests)
        "--env": "env",      # Environment cleanup
        "--all": "all"       # Nuclear option
    }

    sandbox_handler = SandboxHandler()
    return sandbox_handler.destroy_sandbox(mode=mode_map[destruction_type])
```

**Modes**:
- `reset`: Cleans all folders except user/ and tests/
- `env`: Removes .env cache and environment files
- `all`: Nuclear cleanup (requires confirmation)
