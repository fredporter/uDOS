## Technical Highlights

### Lazy Loading Pattern Handling

**Handler Structure:**
```python
class GenerateHandler:
    def __init__(self, viewport=None, logger=None):
        self._gemini_generator = None  # Lazy loaded
        self._vectorizer = None         # Lazy loaded
    
    @property
    def gemini_generator(self):
        if self._gemini_generator is None:
            from core.services.gemini_generator import GeminiGenerator
            self._gemini_generator = GeminiGenerator()
        return self._gemini_generator
```

**Mock Strategy:**
- Patch at import level: `@patch('core.services.gemini_generator.GeminiGenerator')`
- NOT handler attribute: ~~`@patch.object(handler, 'gemini_generator')`~~
- Test lazy loading: `assert handler._gemini_generator is None` before access
