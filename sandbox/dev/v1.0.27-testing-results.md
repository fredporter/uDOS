## Parser Bug Fix (CRITICAL)

**Issue Discovered**: GUIDE and DIAGRAM commands showing `TypeError: replace() argument 2 must be str, not None`

**Root Cause**: `core/uDOS_parser.py` line 71 didn't check for null values in DEFAULT_PARAMS before calling `str.replace()`

**Fix Applied**:
```python
# Before:
if placeholder in ucode:
    ucode = ucode.replace(placeholder, value)

# After:
if placeholder in ucode and value is not None:
    ucode = ucode.replace(placeholder, value)
```

**Impact**: GUIDE and DIAGRAM commands now work correctly
- ✅ GUIDE LIST - Shows available guides
- ✅ DIAGRAM LIST - Shows diagram library status
