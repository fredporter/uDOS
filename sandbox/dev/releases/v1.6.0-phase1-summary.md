### 5. DIAGRAM GENERATE Command
**File:** `core/commands/diagram_handler.py`
**Status:** ✅ Complete
**Added:** 300+ lines

**Multi-Stage Processing:**
1. **Stage A**: Source Analysis - Extract text, images, metadata
2. **Stage B**: Web Crawl - Optional PEEK integration for enrichment
3. **Stage C**: Text Processing - Gemini generates Markdown with citations
4. **Stage D**: Asset Generation - Technical-Kinetic SVG creation
5. **Stage E**: Final Assembly - Combine, validate, save with bibliography

**Command Syntax:**
```bash
DIAGRAM GENERATE <source_file> [options]

Options:
  --crawl              # Enable web crawler (via PEEK)
  --style technical    # SVG style: technical|kinetic|hybrid
  --output <dir>       # Output directory (default: knowledge/generated/)
  --citations-strict   # Require 100% citation coverage
  --validate          # Run full quality validation
  --format md,svg      # Output formats: md,svg,ascii,teletext
  --batch             # Batch process entire directory
```

**Examples:**
```bash
# Single document
DIAGRAM GENERATE sandbox/test_water_purification.md

# With web enrichment
DIAGRAM GENERATE guide.pdf --crawl --style kinetic

# Batch with strict citations
DIAGRAM GENERATE knowledge/fire/ --batch --citations-strict

# Custom output and formats
DIAGRAM GENERATE topic.md --output docs/ --format md,svg,ascii
```

**Output Structure:**
```
knowledge/generated/
├── <filename>.md          # Markdown with citations + bibliography
├── <filename>.svg         # Technical-Kinetic diagram
└── <filename>.meta.json   # Generation metadata & quality metrics
```

## 🎨 Technical-Kinetic SVG Style

### Mandatory Constraints

| Element | Constraint | Examples |
|---------|-----------|----------|
| **Color** | Black (#000) and White (#FFF) ONLY | No grays, gradients, raster |
| **Lines** | 2-3px stroke weight | Clean, consistent |
| **Typography** | Sans-serif (Helvetica, Arial) | 12-16pt headers, 8-10pt labels |
| **Geometry** | MCM: circles, rectangles, clean angles | 45°, 90° preferred |
| **Flow** | Curved conduits, gears, levers | Exaggerated kinetic motion |
| **Shading** | Vector patterns: hatch, stipple, wavy | No raster effects |

### Validation Rules
- ✅ No hex colors except #000000 and #FFFFFF
- ✅ No gradients (linearGradient, radialGradient)
- ✅ No raster images (<image> tags)
- ✅ File size < 50KB
- ✅ Valid SVG structure

---

## 📊 Citation Requirements

### Coverage Targets
- **Standard**: ≥95% of sentences cited
- **Strict Mode**: 100% of sentences cited (--citations-strict)
- **Paragraphs**: 100% must have at least one citation

### Citation Format
```markdown
Water should be boiled for 1 minute.<cite>1</cite> At high altitudes,
boil for 3 minutes.<web>2</web>

## References

### Source Document
1. "Water Purification Manual" by John Doe (2025-11-26)

### Web Sources
2. "CDC Water Treatment Guidelines" <https://cdc.gov/water> (2025)
```

### Validation Output
```json
{

  "coverage": 0.97,
  "min_coverage": 0.95,
  "total_sentences": 30,
  "cited_paragraphs": 8,
  "citations": {
    "document": 15,
    "web": 8,
    "metadata": 2,
    "total": 25
  },
  "issues": []
}
```
