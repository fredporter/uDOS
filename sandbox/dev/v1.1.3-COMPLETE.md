## Usage Examples

```bash
# List all templates
🔮 > MISSION TEMPLATES

# Preview a template
🔮 > MISSION TEMPLATE habit-tracking

# Create mission from template
🔮 > MISSION CREATE --template habit-tracking --id meditation-90 \
       --vars "HABIT_NAME=Morning meditation,TRACKER=Alex,FREQUENCY=Daily,DURATION=90"

# Start the mission
🔮 > MISSION START meditation-90
```
