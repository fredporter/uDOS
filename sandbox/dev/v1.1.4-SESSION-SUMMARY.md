## Command Examples

### Basic Usage
```bash
# Flowchart
DRAW FLOW "Start → Process → End"

# Tree diagram
DRAW TREE "System\nUsers\nData"

# Table
DRAW GRID "Name,Age\nAlice,30\nBob,25"

# Org chart
DRAW HIERARCHY "CEO\n  CTO\n  CFO"

# Auto-detect
DRAW "Process checkout workflow"
```

### Advanced Options
```bash
# File input
DRAW FLOW --file workflow.txt

# Save output
DRAW TREE "Structure" --save diagram.txt

# JSON data
DRAW FLOW --data flowchart.json

# Export format
DRAW GRID "Data" --format unicode

# Combined
DRAW FLOW --file input.txt --save output.txt --format ansi
```
