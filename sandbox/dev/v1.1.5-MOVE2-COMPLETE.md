## Usage Patterns Documented

### Basic Commands
```bash
SVG water filter                          # Simple lineart
SVG fire triangle --style blueprint       # Technical drawing
SVG shelter types --style isometric       # 3D view
SVG campfire --style sketch --save fire.svg  # Save to file
```

### Advanced Usage
```bash
# Process flows
SVG "water purification: collect, filter, store" --style blueprint

# System diagrams
SVG "survival camp: shelter, fire, water, signal" --style isometric

# Comparisons
SVG "knot types: bowline, square, clove hitch" --style sketch
```

### Integration
```bash
# With GUIDE
GUIDE water/purification
SVG "water purification methods" --style blueprint

# With DRAW
DRAW flowchart "water filter"  # ASCII preview
SVG "water filter diagram" --style lineart  # Polished SVG
```

## Known Issues & Mitigations

1. **Gemini API Safety Filters**
   - **Issue**: Some prompts trigger finish_reason=2 (safety block)
   - **Mitigation**: Fallback to template generation
   - **Impact**: Minimal (templates work well)

2. **Post-Processing Complexity**
   - **Issue**: ElementTree adds namespace prefixes
   - **Solution**: Simplified to string-based post-processing
   - **Result**: xmlns and viewBox reliably added

3. **Performance (AI Mode)**
   - **Issue**: AI generation can take 2-5 seconds
   - **Mitigation**: Template fallback is <1ms
   - **Note**: Users can disable preview for faster batch generation
