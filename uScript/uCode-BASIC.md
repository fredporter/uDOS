# uCode BASIC Command Set (Visual Basic–style)

## PRINT
- Syntax: PRINT <string>
- Description: Output a string to the console.

## SET
- Syntax: SET <var> = <value>
- Description: Assign a value to a variable.

## IF / THEN
- Syntax: IF <condition> THEN <command>
- Description: Conditional execution.

## FOR / NEXT
- Syntax: FOR <var> = <start> TO <end> [STEP <n>] ... NEXT
- Description: Looping.

## INCLUDE
- Syntax: INCLUDE <filename>
- Description: Import another script.

## WAIT / SLEEP
- Syntax: WAIT <seconds>
- Description: Pause execution.
