# CAPTURE: topic-template-v1

## Purpose
Capture or enrich a user topic in the local knowledge-tree using the shared
v1.5 Markdown runbook shape.

## Inputs
- Topic: {{topic}}
- Source material: {{source_material}}
- Capture mode: {{capture_mode}}
- Destination path: {{destination_path}}

## Steps
1. Review the topic and source material.
2. Extract or summarize the important facts.
3. Organize the material into local Markdown notes.
4. Record follow-up actions or related workflows if needed.

## Outputs
- {{destination_path}}

## Evidence
- Local topic note created or updated
- Source references recorded
- Follow-up actions listed

## Notes
- Duplicate seed material before editing when starting from global templates.
- Keep user-specific enrichment in the local knowledge-tree.
