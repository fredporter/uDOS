# Migration Guide: v1.4 → v1.5

## Major Changes
- Ollama removed
- GPT4All standardized as local assist
- Wizard Network exclusive API routing
- Advanced budget tiers enforced

## Required Actions
1. Remove Ollama dependencies
2. Install GPT4All local runtime
3. Configure Wizard Network budget policies
4. Update workflow templates to v1.5 schema

## Optional Enhancements
- Enable deferred execution queue
- Enable budget auto-cooldown windows
