# SUBMISSION: device-record-template-v1

## Purpose
Prepare a device or tool record submission using the shared v1.5 Markdown
template shape.

## Inputs
- Record name: {{record_name}}
- Category: {{category}}
- Source material: {{source_material}}
- Submission path: {{submission_path}}

## Steps
1. Gather the record details and local evidence.
2. Draft the submission in Markdown.
3. Mark any uncertain fields for contributor review.
4. Store the submission in the local writable tree.

## Outputs
- {{submission_path}}

## Evidence
- Submission draft created
- Supporting references attached
- Review status recorded
