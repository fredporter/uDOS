# Mistral Vive in the `@dev` Workspace

Updated: 2026-03-05
Status: short public usage note

`Mistral Vive` is available only in the `@dev` workspace as a contributor tool lane.

## Scope

- Use standard `ucode` runtime for normal operator workflows.
- Use `Mistral Vive` only for contributor development tasks inside `@dev`.
- Keep runtime contracts and specs in `docs/`; keep contributor implementation detail in `dev/docs/`.

## Quick Usage

1. Ensure Dev Mode lane is active in Wizard.
2. Open `@dev` workspace.
3. Run contributor operations through the Dev Mode tool surface.
