## 🚀 Quick Start Guide

### **New Users - Start Here!**

**1. Read the User Guide**
```bash
cat docs/USER-GUIDE.md
```

**2. Learn the Naming Convention**
```bash
cat docs/Style-Guide.md
```

**3. Explore Features**
```bash
# Key commands to get started
STATUS          # System overview
HELP            # Command reference  
MEMORY LIST     # List your files
DASH            # Dashboard view
```

### **Developers**

**1. Development Environment**
```bash
# Setup wizard development mode
cd uDEV/
./tools/dev-session-logger.sh start "My Project" "Description"
```

**2. Technical Documentation**
```bash
# Read system architecture
cat docs/ARCHITECTURE.md

# Check technical docs
ls uCORE/docs/
```

### **Content Creators**

## 🌟 What's New in v1.3

### **Major Features**

#### 🕒 **Timezone Integration**
- 38 timezone codes mapped from city dataset
- Automatic timezone detection for file naming
- Global compatibility with standard formats

#### 🧙‍♂️ **Wizard Development Mode (uDEV)**
- Exclusive development environment for advanced users
- Automated session logging with uLOG format
- VS Code integration with workspace settings
- Task management with ASSIST mode integration

#### 🤖 **Enhanced ASSIST Mode**
- Sandbox task management for AI workflows
- Natural language processing for task creation
- Context-aware assistance for development
- Automated documentation generation

#### 📏 **CAPS-NUMERIC-DASH Naming**
- Strict naming convention for all files
- Improved organization and searchability
- Consistent formatting across components
- Better automation support

#### 📚 **Documentation Consolidation**
- User documentation in root `docs/` folder
- Technical documentation in `uCORE/docs/`
- Comprehensive cross-references
- Clear separation of concerns
