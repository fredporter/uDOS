## 🚀 Quick Access

### Essential Commands Reference
```bash
# View documentation
./uCORE/code/ucode.sh [HELP]                    # General help
./uCORE/code/ucode.sh [HELP] [MEMORY]           # Specific command help
./uCORE/code/ucode.sh "[DASHBOARD] SHOW"        # Quick system overview

# Documentation tools
./uCORE/code/ucode.sh "[DOCS] SEARCH keyword"   # Search documentation
./uCORE/code/ucode.sh "[DOCS] INDEX"            # Documentation index
./uCORE/code/ucode.sh "[DOCS] RECENT"           # Recently updated docs
```

### Common Learning Patterns
```bash
# New user onboarding
1. Read user-guides/USER-GUIDE.md
2. Practice with user-guides/uScript-VB-Commands.md
3. Explore technical/uDOS-Concepts-v1.3.md
4. Reference as needed from reference/

# Developer onboarding
1. Read development/uCode-Developer-Guide.md
2. Setup with development/VS-Code-Dev-Mode-Guide.md
3. Study technical/ARCHITECTURE.md
4. Follow standards in reference/Template-Standard.md
```
