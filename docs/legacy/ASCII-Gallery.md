## Usage Guidelines

### Color Coordination
- **System Elements**: Use cyan and blue tones
- **Success States**: Green highlights and borders
- **Warning States**: Yellow/orange accents
- **Error States**: Red highlights (sparingly)
- **Interactive Elements**: Purple/magenta for buttons

### Spacing Standards
- **Panel Padding**: 1-2 characters inside borders
- **Section Spacing**: 1 blank line between sections
- **Element Spacing**: Consistent alignment and spacing

### Accessibility
- **High Contrast**: Ensure elements are visible in B&W
- **Clear Hierarchy**: Use borders and spacing for organization
- **Consistent Symbols**: Stick to standard Unicode characters

### Performance
- **Lightweight**: Keep ASCII art efficient to render
- **Caching**: Reuse common elements and patterns
- **Responsive**: Adapt to different terminal sizes
