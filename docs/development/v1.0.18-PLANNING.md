# v1.0.18 Development Planning - Apocalypse Adventures & Experience System

**Version**: v1.0.18
**Codename**: Apocalypse Adventures
**Status**: 📋 Planning Phase
**Target Date**: January 2026
**Focus**: Survival-themed adventures with experience points, offline-first desktop server, and practical knowledge integration

---

## 🎯 Vision

Transform uDOS into a survival companion that teaches real-world skills through interactive apocalypse scenarios. Combine entertainment (NetHack-style gameplay) with education (practical survival knowledge) in an offline-first system that works when you need it most.

**Core Philosophy**:
- **Learn by Playing**: Survival skills taught through interactive adventures
- **Offline-First**: Works WITHOUT internet when emergencies strike
- **Practical Over Political**: Real skills (shelter, water, medicine) not ideologies
- **No Wealth Accumulation**: XP shows competence, barter replaces currency
- **Real-World Application**: Every adventure teaches actionable survival knowledge

## 📊 Feature Breakdown

### Phase 1: Experience Point System (Week 1)
**Goal**: Implement XP tracking and skill progression system

#### Features
1. **XP Engine** (`core/services/xp_service.py`)
   - Track XP across categories: Usage, Information, Contribution, Connection
   - Skill trees: Shelter, Food, Water, Medicine, Defense, Tools
   - Level progression formulas (exponential growth)
   - Achievement unlock system
   - Persistent XP storage (SQLite)

2. **XP Commands** (`core/commands/xp_handler.py`)
   - `XP STATUS` - Show total XP and level
   - `XP BREAKDOWN` - Category breakdown
   - `XP SKILLS` - Show skill tree progress
   - `XP ACHIEVEMENTS` - List unlocked achievements
   - `XP HISTORY` - Recent XP gains

3. **XP Integration**
   - Award XP for command usage
   - Award XP for reading knowledge content
   - Award XP for completing adventures
   - Award XP for contributing knowledge

### Phase 3: Survival Variables & State Management (Week 3)
**Goal**: Extend uCODE variable system for survival scenarios

#### Features
1. **Survival Variable Types** (`core/uDOS_ucode.py`)
   - **Resource variables**: `{food}`, `{water}`, `{medicine}`, `{fuel}`
   - **Skill variables**: `{shelter_skill}`, `{medical_skill}`, `{defense_skill}`
   - **Health variables**: `{health}`, `{energy}`, `{morale}`, `{injuries}`
   - **Time variables**: `{day}`, `{hour}`, `{season}`, `{countdown}`
   - **Location variables**: `{current_zone}`, `{safe_zones}`, `{danger_areas}`
   - **Relationship variables**: `{trust_level}`, `{allies}`, `{threats}`

2. **State Persistence** (`core/services/scenario_service.py`)
   - Save scenario state to SQLite
   - Load scenario state on resume
   - Checkpoint system (manual save points)
   - Auto-save on critical events

3. **Survival Script Commands** (extend `core/uDOS_ucode.py`)
   - `[SURVIVAL|RESOURCE*<action>*<type>*<amount>]` - Manage resources
   - `[SURVIVAL|SKILL*<action>*<skill>*<value>]` - Skill checks/updates
   - `[SURVIVAL|HEALTH*<action>*<type>*<value>]` - Health management
   - `[SURVIVAL|TIME*<action>*<value>]` - Time tracking
   - `[SURVIVAL|LOCATION*<action>*<zone>]` - Zone management
   - `[SURVIVAL|THREAT*<action>*<type>*<level>]` - Threat handling

### Phase 5: Pre-Built Apocalypse Scenarios (Week 5)
**Goal**: Create 8 complete survival scenarios

#### Scenarios

1. **The Last Day** (30 min, Beginner)
   - **Theme**: Zombie outbreak in city
   - **Objective**: Escape to countryside safe zone
   - **Skills**: Decision-making, resource management
   - **Knowledge**: Evacuation planning, emergency supplies
   - **Map**: 40x20 city grid with safe zones
   - **XP Rewards**: 100 base + 50 per objective

2. **Nuclear Winter** (2 hours, Intermediate)
   - **Theme**: Post-nuclear survival
   - **Objective**: Build shelter, gather supplies, survive 30 days
   - **Skills**: Shelter building, radiation safety, rationing
   - **Knowledge**: Fallout protection, water purification
   - **Map**: 60x30 wasteland with bunkers and resources
   - **XP Rewards**: 500 base + skill progression

3. **Grid Down** (Ongoing campaign, Advanced)
   - **Theme**: Power grid collapse
   - **Objective**: Transition to off-grid living
   - **Skills**: Food production, water systems, energy generation
   - **Knowledge**: Solar power, gardening, well construction
   - **Map**: 80x40 rural area with homestead
   - **XP Rewards**: Progressive (daily survival = 50 XP)

4. **The Collapse** (Campaign, Advanced)
   - **Theme**: Economic meltdown
   - **Objective**: Build barter network, establish community
   - **Skills**: Trading, networking, resource allocation
   - **Knowledge**: Barter economy, skill sharing, mutual aid
   - **Map**: 100x50 town with multiple communities
   - **XP Rewards**: Trade-based (successful trades = XP)

5. **Pandemic Protocol** (1 hour, Intermediate)
   - **Theme**: Disease outbreak
   - **Objective**: Avoid infection, treat sick, find cure info
   - **Skills**: Medical triage, hygiene, quarantine
   - **Knowledge**: Disease prevention, herbal medicine
   - **Map**: 50x25 medical facility and surrounding area
   - **XP Rewards**: 300 base + patient saves

6. **Rammed Earth Shelter** (30 min, Tutorial)
   - **Theme**: Emergency housing construction
   - **Objective**: Build shelter from scratch using PDF guides
   - **Skills**: Construction, material sourcing
   - **Knowledge**: Rammed earth technique, cob mixing
   - **Map**: 30x15 construction site
   - **XP Rewards**: 150 base + quality bonus

7. **Water Purification Quest** (45 min, Beginner)
   - **Theme**: Find and purify water sources
   - **Objective**: Secure 7 days of clean water
   - **Skills**: Water collection, filtration, testing
   - **Knowledge**: Purification methods, storage
   - **Map**: 40x20 area with water sources
   - **XP Rewards**: 200 base + safety bonus

8. **Herbal Medicine** (1 hour, Intermediate)
   - **Theme**: Natural remedy gathering
   - **Objective**: Identify and use 10 medicinal plants
   - **Skills**: Plant identification, remedy preparation
   - **Knowledge**: Herbalism, natural first aid
   - **Map**: 50x25 wilderness with plant locations
   - **XP Rewards**: 250 base + correct identification

#### Success Criteria
- [ ] 8 scenarios created with full content
- [ ] Each scenario tested for completeness
- [ ] XP rewards balanced across scenarios
- [ ] Knowledge content embedded
- [ ] All maps generated and functional

### Phase 7: Offline Desktop Server (Week 7)
**Goal**: Create standalone desktop application for offline survival hub

#### Features
1. **Desktop Application** (`core/uDOS_desktop.py`)
   - Electron/Tauri wrapper (or Python PyWebView)
   - System tray icon
   - Always-on background mode
   - Minimal resource usage
   - Auto-start on boot option

2. **SURVIVE Command** (`core/commands/survive_handler.py`)
   - `SURVIVE DESKTOP` - Launch desktop survival hub
   - `SURVIVE STATUS` - Check desktop server status
   - `SURVIVE SHUTDOWN` - Close desktop server
   - `SURVIVE TRAY` - Minimize to system tray

3. **Offline Web UI** (`extensions/survival_desktop/`)
   - Survival dashboard (health, resources, threats)
   - Interactive map interface
   - Inventory manager
   - Skill tree visualization
   - Journal system (log entries)
   - Quick actions (common survival tasks)
   - Dark apocalypse themes (zombie, wasteland, bunker)

4. **Zero External Dependencies**
   - All data local (SQLite)
   - No cloud services
   - No internet required
   - Bundled assets only
   - Self-contained executable

5. **Backup Systems**
   - Auto-backup every 4 hours
   - Manual backup command
   - Multiple backup locations
   - Backup restoration
   - Export to USB drive
