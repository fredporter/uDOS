# Basic Gemini Profile

**Simple Google Gemini CLI Integration**

## Configuration

```json
{
  "profile_name": "basic_gemini",
  "mode": "assist",
  "context_awareness": true,
  "project_integration": true,
  "response_style": "helpful",
  "technical_level": "adaptive"
}
```

## Usage Patterns

### ASSIST Mode
- Development guidance and code assistance
- Architecture recommendations
- Debugging help
- Best practices suggestions

### Context Integration
- Current project awareness
- Mission status understanding
- User role adaptation
- System state knowledge

## Response Style
- Clear and concise explanations
- Technical accuracy
- Context-appropriate detail level
- Action-oriented suggestions

## System Integration
- uDOS command generation
- Template system awareness
- File structure understanding
- Workflow optimization
