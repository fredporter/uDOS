## Quick Start Guide

### Installation (5 minutes)

```ascii
┌─── QUICK INSTALLATION ──────────────────────────────┐
│                                                     │
│  Step 1: Download                                   │
│  ──────────────────                                 │
│  git clone <repository-url>                         │
│  cd uDOS                                            │
│                                                     │
│  Step 2: Setup                                      │
│  ─────────────                                      │
│  chmod +x install/setup.sh                          │
│  ./install/setup.sh                                 │
│                                                     │
│  Step 3: Launch                                     │
│  ──────────────                                     │
│  ./uCode/ucode.sh                                   │
│                                                     │
│  Step 4: Explore                                    │
│  ───────────────                                    │
│  Follow the interactive tutorial!                   │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### First Commands

```bash
STATUS          # System overview
HELP            # Command reference  
GO              # Browse shortcuts
[MEM|LIST]      # List your files
MISSION CREATE "My First Task"  # Create a mission
DASH            # Dashboard view
TUTORIAL        # Replay the adventure
```
