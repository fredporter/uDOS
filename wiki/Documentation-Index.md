### Markdown Features
- **Bold** - Important terms, file names
- `Code` - Commands, code snippets, file paths
- > Blockquotes - Notes, warnings, tips
- ⭐ - New in current version
- ✅ - Complete/working
- ⚠️ - Warning/deprecated
- 🔧 - Development/technical
- 📖 - Documentation/reference
