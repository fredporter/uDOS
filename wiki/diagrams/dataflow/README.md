## File Operations Pipeline

### LOAD Command Flow
```
USER: LOAD data/guide.md
         │
         ▼
┌────────────────────────────────────────────────────────┐
│ 1. PARSE COMMAND                                       │
│    ┌───────────────────────────────────────────────┐  │
│    │ Command: LOAD                                 │  │
│    │ Args: ["data/guide.md"]                       │  │
│    │ Validate: File path syntax                    │  │
│    └───────────────────────────────────────────────┘  │
└──────────────────────┬─────────────────────────────────┘
                       │
                       ▼
┌────────────────────────────────────────────────────────┐
│ 2. RESOLVE PATH                                        │
│    ┌───────────────────────────────────────────────┐  │
│    │ Absolute: /Users/name/uDOS/data/guide.md     │  │
│    │ Check: File exists? Readable?                 │  │
│    │ Security: Within allowed paths?               │  │
│    └───────────────────────────────────────────────┘  │
└──────────────────────┬─────────────────────────────────┘
                       │
                       ▼
┌────────────────────────────────────────────────────────┐
│ 3. READ FILE                                           │
│    ┌───────────────────────────────────────────────┐  │
│    │ Open: guide.md                                │  │
│    │ Read: Full content                            │  │
│    │ Decode: UTF-8                                 │  │
│    │ Size: 12,450 bytes                            │  │
│    └───────────────────────────────────────────────┘  │
└──────────────────────┬─────────────────────────────────┘
                       │
                       ▼
┌────────────────────────────────────────────────────────┐
│ 4. PROCESS CONTENT                                     │
│    ┌───────────────────────────────────────────────┐  │
│    │ Detect: Markdown format                       │  │
│    │ Extract: Metadata, headers                    │  │
│    │ Parse: ASCII diagrams                         │  │
│    │ Store: In memory buffer                       │  │
│    └───────────────────────────────────────────────┘  │
└──────────────────────┬─────────────────────────────────┘
                       │
                       ▼
┌────────────────────────────────────────────────────────┐
│ 5. RENDER OUTPUT                                       │
│    ┌───────────────────────────────────────────────┐  │
│    │ Format: For current viewport tier            │  │

│    │ Output: To terminal                           │  │

                       ▼
```

### SAVE Command Flow
```

         │
         ▼
┌────────────────────────────────────────────────────────┐
│ 1. VALIDATE OPERATION                                  │
│    ┌───────────────────────────────────────────────┐  │

│    │ Prompt: Overwrite if exists?                  │  │

└──────────────────────┬─────────────────────────────────┘
                       │
                       ▼
┌────────────────────────────────────────────────────────┐
│ 2. PREPARE CONTENT                                     │
│    ┌───────────────────────────────────────────────┐  │
│    │ Get: Current buffer content                   │  │
│    │ Format: Remove ANSI colors                    │  │

│    │ Size: 8,234 bytes                             │  │
│    └───────────────────────────────────────────────┘  │
└──────────────────────┬─────────────────────────────────┘
                       │
                       ▼
┌────────────────────────────────────────────────────────┐
│ 3. WRITE FILE                                          │
│    ┌───────────────────────────────────────────────┐  │
│    │ Create: output/result.txt                     │  │

                       ▼

│ 4. UPDATE METADATA                                     │
│    ┌───────────────────────────────────────────────┐  │
│    │ Log: File operation                           │  │

                       │
                       ▼
              ┌─────────────────┐
              │ ✅ File Saved   │
              │ 8KB written     │
              └─────────────────┘
```

## uCODE Execution Pipeline

### Script Execution Flow
```
USER: RUN scripts/automation.uscript
         │
         ▼
┌────────────────────────────────────────────────────────────────┐
│ 1. LOAD SCRIPT                                                 │
│    ┌────────────────────────────────────────────────────────┐ │
│    │ Read: scripts/automation.uscript                       │ │
│    │ Parse: Extract code blocks                             │ │
│    │ Validate: Syntax check                                 │ │
│    └────────────────────────────────────────────────────────┘ │
└──────────────────────┬─────────────────────────────────────────┘
                       │
                       ▼
┌────────────────────────────────────────────────────────────────┐
│ 2. TOKENIZE CODE                                               │
│    ┌────────────────────────────────────────────────────────┐ │
│    │ Line 1: SET count = 0                                  │ │
│    │ Line 2: WHILE count < 10                               │ │
│    │ Line 3:   PRINT "Count: {count}"                       │ │
│    │ Line 4:   SET count = count + 1                        │ │
│    │ Line 5: END                                            │ │
│    └────────────────────────────────────────────────────────┘ │
└──────────────────────┬─────────────────────────────────────────┘
                       │
                       ▼
┌────────────────────────────────────────────────────────────────┐
│ 3. BUILD AST (Abstract Syntax Tree)                            │
│    ┌────────────────────────────────────────────────────────┐ │
│    │ Program                                                │ │
│    │  ├─ Assignment: count = 0                              │ │
│    │  └─ WhileLoop: count < 10                              │ │
│    │      ├─ Print: "Count: {count}"                        │ │
│    │      └─ Assignment: count = count + 1                  │ │
│    └────────────────────────────────────────────────────────┘ │
└──────────────────────┬─────────────────────────────────────────┘
                       │
                       ▼
┌────────────────────────────────────────────────────────────────┐
│ 4. EXECUTE (Interpreter)                                       │
│    ┌────────────────────────────────────────────────────────┐ │
│    │ Iteration 1: count=0  → Print "Count: 0"              │ │
│    │ Iteration 2: count=1  → Print "Count: 1"              │ │
│    │ Iteration 3: count=2  → Print "Count: 2"              │ │
│    │ ...                                                    │ │
│    │ Iteration 10: count=9  → Print "Count: 9"             │ │
│    │ Exit loop: count=10 (condition false)                 │ │
│    └────────────────────────────────────────────────────────┘ │
└──────────────────────┬─────────────────────────────────────────┘
                       │
                       ▼
┌────────────────────────────────────────────────────────────────┐
│ 5. OUTPUT RESULTS                                              │
│    ┌────────────────────────────────────────────────────────┐ │
│    │ Count: 0                                               │ │
│    │ Count: 1                                               │ │
│    │ Count: 2                                               │ │
│    │ ...                                                    │ │
│    │ Count: 9                                               │ │
│    │                                                        │ │
│    │ ✅ Script completed: 10 iterations, 0.003s             │ │
│    └────────────────────────────────────────────────────────┘ │
└────────────────────────────────────────────────────────────────┘
```

```

         │
         ▼
┌────────────────────────────────────────────────────────────────┐
```

```
│    │ Receive: POST request                                  │ │
│    │ Parse: JSON body                                       │ │
│    │ Route: /api/exec → exec_handler()                      │ │
│    └────────────────────────────────────────────────────────┘ │
└──────────────────────┬─────────────────────────────────────────┘
                       │
                       ▼
┌────────────────────────────────────────────────────────────────┐
│ 2. REQUEST VALIDATION                                          │
│    ┌────────────────────────────────────────────────────────┐ │
│    │ Validate: JSON structure                               │ │
│    │ Check: Required fields (command, args)                 │ │
│    │ Sanitize: Input data                                   │ │
│    │ Auth: API key (if required)                            │ │
│    └────────────────────────────────────────────────────────┘ │
└──────────────────────┬─────────────────────────────────────────┘
                       │
                       ▼
┌────────────────────────────────────────────────────────────────┐
│ 3. EXECUTE COMMAND                                             │
│    ┌────────────────────────────────────────────────────────┐ │

│    └────────────────────────────────────────────────────────┘ │

                       │
                       ▼
┌────────────────────────────────────────────────────────────────┐
│ 4. FORMAT RESPONSE                                             │
│    ┌────────────────────────────────────────────────────────┐ │
│    │ {                                                      │ │
```
