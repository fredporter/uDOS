## Overview

Australia has the world's most venomous snakes. Proper first aid is critical - inappropriate treatment kills more people than snake venom.

## 1. Pressure Immobilisation Technique (PIT)

**CRITICAL:** Australia's unique first aid - DO NOT use for most other countries

### Step-by-Step Application

```
Step 1: Keep patient COMPLETELY STILL
┌───────────────────────────────────────┐
│                                       │
│  DO NOT wash bite                     │
│  DO NOT cut bite                      │
│  DO NOT apply tourniquet              │
│  DO NOT suck venom                    │
│                                       │
│  ╔═══════════════════════════════╗    │
│  ║  Patient must remain CALM     ║    │
│  ║  and NOT MOVE affected limb   ║    │
│  ╚═══════════════════════════════╝    │
│                                       │
└───────────────────────────────────────┘

Step 2: Mark bite location
        ▲
        │ Mark with pen/texta
        │ (for hospital venom ID)
        ●══════════ Bite site
       ╱│╲
      ╱ │ ╲  DO NOT WASH
     ╱  │  ╲ Venom traces needed
    ════●════ for identification

Step 3: Apply bandage START at bite
    Pressure Bandage Application:

    First wrap (over bite):
    ╔═════════════╗
    ║▓▓▓▓▓▓▓▓▓▓▓▓▓║ ← Firm pressure
    ║░░●░Bite░░░░░║    (not tight!)
    ║▓▓▓▓▓▓▓▓▓▓▓▓▓║
    ╚═════════════╝

    Continue wrapping (overlap 50%):
    ╔═══════════════════════════╗
    ║▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓║
    ║▓▓●═════════════════════▓▓║ ← Entire limb
    ║▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓║   covered
    ╚═══════════════════════════╝

    Correct tightness:
    Should fit like firm sock/stocking
    Finger should just slide under
    NOT a tourniquet (don't cut circulation)

Step 4: Splint limb
    Arm bite:
    ╔════════════════════════════╗
    ║ ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓  ║ ← Bandaged
    ║ ══════════════════════════ ║ ← Splint (stick)
    ║ ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓  ║
    ╚════════════════════════════╝
    Secure with bandage/tape

    Leg bite:
    ╔════════════════════════════╗
    ║ ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓  ║
    ║ ══════════════════════════ ║ ← Tree branch
    ║ ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓  ║
    ║ ══════════════════════════ ║ ← Both sides
    ║ ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓  ║
    ╚════════════════════════════╝
    Immobilise completely
```

### Bandage Materials

```
Best to Worst:
1. Crepe bandage (10cm × 4.5m)  ████████
2. Compression bandage          ████████
3. Torn clothing strips         ██████
4. Pantyhose/stockings         ████
5. Shirt/towel strips          ██

DO NOT USE:
✗ Rope (too tight)
✗ Wire (cuts circulation)
✗ Belt (tourniquet effect)
✗ Plastic wrap (no pressure)
```

## 4. Symptom Recognition

### Envenomation Symptoms Timeline

```
Brown Snake Venom (Neurotoxic + Coagulopathy):

0-15 min:  ░░░░░░░░░░░░░░░░░░░░░░░░░░
           Headache, nausea
           Abdominal pain
           Vomiting

15-60 min: ▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒
           Drowsiness, confusion
           Difficulty swallowing
           Double vision

1-3 hours: ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓
           Paralysis beginning
           Breathing difficulty
           Blood clotting failure

3-6 hours: ████████████████████████████
           Respiratory failure
           Internal bleeding
           CRITICAL - needs ventilation

Tiger Snake Venom (Myotoxic + Neurotoxic):

0-30 min:  ░░░░░░░░░░░░░░░░░░░░░░░░░░
           Local pain, swelling
           Sweating, vomiting

30-90 min: ▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒
           Muscle weakness
           Paralysis starts feet
           Dark urine (myoglobin)

2-4 hours: ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓
           Ascending paralysis
           Kidney damage
           Breathing difficulty

4-8 hours: ████████████████████████████
           Complete paralysis
           Renal failure
           Respiratory arrest
```

## 7. Snake Awareness Tips

### What TO DO

```
✓ Make noise when bushwalking (stomp, talk)
✓ Use stick to tap ground ahead
✓ Stay on clear paths
✓ Check before sitting on logs
✓ Shake out shoes/clothing before use
✓ Keep grass short around home
✓ Remove rubbish/debris (snake habitat)
✓ Keep torch handy at night
✓ Teach children snake awareness
✓ Have snakebite kit in car/home
```

### What NOT TO DO

```
✗ Walk barefoot in grass
✗ Put hands in holes/crevices
✗ Step over logs (step ON them)
✗ Approach or corner snake
✗ Try to kill/catch snake
✗ Use headphones in bush (can't hear warnings)
✗ Walk through tall grass
✗ Disturb snake habitat
✗ Assume snake is dead (can still bite)
✗ Handle dead snake (venom still active)
```

## 9. Emergency Contacts (State-Specific)

### Triple Zero Information

```
When calling 000 for snakebite:

1. Stay calm, speak clearly
2. "Ambulance - snakebite emergency"
3. Location:
   - Street address OR
   - GPS coordinates OR
   - Grid reference (e.g., "M370 × 185") OR
   - Nearest landmark + distance
4. Patient condition:
   - Conscious? Breathing?
   - Symptoms observed
   - PIT applied? (Yes/No)
5. Stay on line for instructions

Emergency+ App:
- Shows GPS coordinates
- Shows address (if available)
- One-tap to call 000
- Works offline with GPS
```
