## 1. Bowline (Loop Knot)

**Use:** Creates fixed loop that won't slip. "King of knots"

**Strength:** 100% (strongest loop knot)

### Step-by-Step Diagram

```
Step 1: Make overhand loop (rabbit hole)
        │
        │
    ┌───┴───┐
    │   ○   │  ← Loop (rabbit hole)
    └───┬───┘
        │
        │
     Standing
      Part

Step 2: Thread working end up through loop
        │
        ↑  Working
    ┌───┴───┐    end (rabbit)
    │   │   │
    └───┬───┘
        │
        │

Step 3: Around standing part (tree)
        ╭───╮
        │   │←─ Around
    ┌───┴───┐
    │   │   │
    └───┬───┘
        │

Step 4: Back down through loop (hole)
        ↓
    ┌───┴───┐
    │   ●───┼─→ Tighten
    └───┬───┘
        │

Final: Fixed loop that won't slip
        ╭───╮
        │   │
    ┌───┴───┐
    ●═══════● ← Secure loop
    └───────┘
```

### Mnemonic

"Rabbit comes out of the hole, around the tree, and back down the hole"

### Uses

✓ Rescue (body harness)
✓ Securing boat to dock
✓ Hoisting objects
✓ Creating adjustable loop
✓ Climbing anchor (not primary)

## 2. Square Knot (Reef Knot)

**Use:** Join two ropes of equal thickness

**Strength:** 45% (weakens under load)

### Diagram

```
Step 1: Right over left
    ════╗
        ║
    ════╝

Step 2: Left over right
    ╔════
    ║
    ╚════

Complete:
    ╔════╗
    ║    ║
    ╚════╝

View from side:
    ═══▓▓▓▓═══
       ████
    ═══▓▓▓▓═══

CORRECT (square):
    ░░░░
    ████  ← Symmetrical
    ░░░░

WRONG (granny):
    ░░░
     ████  ← Lopsided
      ░░░
```

### Warning

⚠️ **NOT for critical loads**
⚠️ **Can slip with smooth rope**
⚠️ **Never use for climbing**

### Uses

✓ Bandages and first aid
✓ Tying packages
✓ Temporary rope joining
✓ Sail reefing (marine)

## 4. Taut-Line Hitch

**Use:** Adjustable loop for tent lines

**Strength:** 85% (holds under tension)

### Diagram

```
Step 1: Three wraps around standing part
        ║
    ╭───╬───╮
    │   ║   │ ← Wrap 1
    ├───╬───┤
    │   ║   │ ← Wrap 2
    ├───╬───┤
    │   ║   │ ← Wrap 3
    └───╬───┘
        ║

Step 2: One wrap above
        ║
    ┌───╬───┐ ← Wrap 4
    │   ║   │
    ╞═══╬═══╡ ← Coils below
    │   ║   │
    └───╬───┘
        ║

Adjustable:
    ▲ Slide up
    │
    ┼═══┼  ← Grip here
    │
    ▼ Slide down
```

### Adjusting

**Tighten:** Push knot toward anchor
**Loosen:** Pull knot away from anchor
**Lock:** Pull tension on standing part

### Uses

✓ Tent guy lines (essential)
✓ Tarp edges
✓ Clotheslines
✓ Any adjustable tension need

## 5. Figure 8 Loop

**Use:** Strong fixed loop for climbing

**Strength:** 100% (strongest knot)

### Diagram

```
Step 1: Make figure 8
        │
        ○
       ╱ ╲
      8   ○
       ╲ ╱
        ○
        │

Step 2: Thread through and retrace
        ╭─╮
        │ │
    ╭───○─○───╮
    │   ║ ║   │
    │   ○═○   │  ← Follow path
    │   ║ ║   │
    ╰───○─○───╯
        │ │
        ╰─╯

Final: Double figure 8
    ╔═══╗
    ║   ║
    ╚═╗ ╔═╝
      ║ ║
    ╔═╝ ╚═╗
    ║     ║
    ╚═════╝
```

### Inspection

Check for proper "figure 8" shape:
```
Correct:         Wrong:
    ∞                ∝
  Clear 8        Tangled
```

### Uses

✓ **Climbing anchor** (primary use)
✓ Rescue operations
✓ Heavy load lifting
✓ Critical safety applications

## 8. Trucker's Hitch

**Use:** Mechanical advantage for tightening

**Strength:** 300% advantage (3:1 ratio)

### Diagram

```
Step 1: Fixed loop
    ┌───────┐
    │   ╭─╮ │
    │   │○│ │  ← Loop
    │   ╰─╯ │
    └───────┘
        ║

Step 2: Thread through anchor
        ║
    ╭───●───╮
    │   ║   │
    │   ▼   │
    └───○───┘
      Anchor

Step 3: Pull for tension
        ▲
        ║ Pull
    ╭───●
    │   ║
    │   ║  ← 3:1 advantage
    └───○

Final secure:
        ▲
        ╬  ← Half hitches
    ╭───●
    │  ║║
    └──○○
```

### Mechanical Advantage

```
    You pull    Rope tightens
      1 lb    →    3 lbs
     10 lbs   →   30 lbs
    100 lbs   →  300 lbs
```

### Uses

✓ **Securing cargo** (primary use)
✓ Tarp tensioning
✓ Hauling heavy loads
✓ Tightening tie-downs

**Strength:** 65% (very secure)

```
╔══════════════════════════╦═══════════════════════════╗
║ Situation                ║ Best Knot                 ║
╠══════════════════════════╬═══════════════════════════╣
║ Fixed loop               ║ Bowline or Figure 8       ║
║ Adjustable loop          ║ Taut-Line Hitch           ║
║ Join same-size ropes     ║ Double Fisherman's        ║
║ Join different ropes     ║ Sheet Bend                ║
║ Attach to pole/tree      ║ Clove Hitch               ║
║ Tighten cargo            ║ Trucker's Hitch           ║
║ Climbing (critical)      ║ Figure 8 Loop             ║
║ Mid-line loop            ║ Butterfly Loop            ║
║ Lift barrel              ║ Barrel Hitch              ║
║ Slide on rope            ║ Prusik Knot               ║
```

## Rope Care

### Inspect Before Use

```
Good Rope:         Bad Rope:
════════          ═══╱╲═══  ← Frayed
Smooth            ═══  ═══  ← Broken
════════          ═══▓▓═══  ← Dirty/degraded
Uniform           ═══╳╳═══  ← Damaged
```

### Storage

✓ Cool, dry location
✓ Coiled properly (no kinks)
✓ Away from UV light
✓ Protected from chemicals
✓ Inspect regularly
