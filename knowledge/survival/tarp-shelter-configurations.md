# Tarp Shelter Configurations

**Category**: Shelter & Building
**Difficulty**: Beginner to Intermediate
**Time Required**: 15-45 minutes
**Materials**: Tarp, cordage, stakes/anchor points
**Season**: All seasons (configuration varies)

## Overview

A quality tarp and knowledge of configurations provides versatile, quick shelter in any environment. Faster than natural shelters, lighter than tents, adaptable to terrain and weather.

## Tarp Selection

### Size Guidelines

**Solo**:
- Minimum: 6' x 8' (cramped)
- Recommended: 8' x 10'
- Spacious: 10' x 12'

**Two Person**:
- Minimum: 8' x 10'
- Recommended: 10' x 12'
- Spacious: 12' x 12'

**Group/Gear**:
- 12' x 12' or larger
- Multiple tarps better than one huge one

### Materials

**Silnylon**:
- Lightest weight
- Packs small
- Water-resistant (not waterproof)
- Stretches when wet
- Expensive

**Silpoly**:
- Lightweight
- Doesn't stretch when wet
- Waterproof with seam sealing
- Good balance of features

**Polyethylene (Poly)**:
- Common blue tarps
- Heavy but cheap
- Fully waterproof
- Noisy in wind
- Degrades in UV over time

**Canvas**:
- Heavy
- Breathable
- Durable
- Fire-resistant (treated)
- Traditional/bushcraft choice

### Features

**Grommets**:
- Metal eyelets at edges/corners
- Minimum every 3 feet
- Check for reinforcement
- More = more setup options

**Reinforced Corners**:
- Double/triple layer
- Prevents tearing at stress points
- Essential for longevity

**Color**:
- Bright (orange/yellow): High visibility, rescue situations
- Earth tones (green/brown): Low profile, blends in
- Reflective (silver): Heat reflection in desert

## Essential Knots & Techniques

### Trucker's Hitch (Tensioning)

Creates 3:1 mechanical advantage for tight lines:

```
1. Tie fixed end to anchor
2. Create loop 2/3 way along rope
3. Pass working end through anchor
4. Thread through loop (pulley effect)
5. Pull tight
6. Secure with half hitches
```

**Use**: Ridgeline, guy lines needing tension

### Taut-Line Hitch (Adjustable)

Slides to adjust, holds under tension:

```
1. Wrap around anchor/stake
2. Make 2-3 wraps around standing line (toward load)
3. Make 1 wrap on other side
4. Tighten against load
```

**Use**: Adjustable guy lines, changing tension

### Clove Hitch (Quick Attachment)

Fast, temporary attachment:

```
1. Make loop around pole
2. Make second loop crossing first
3. Tighten
```

**Use**: Attaching to trees, poles, quick setups

### Bowline (Fixed Loop)

Creates non-slip loop at end of rope:

```
"Rabbit hole" method:
1. Make small loop (rabbit hole)
2. Pass end up through loop (rabbit out)
3. Around standing line (around tree)
4. Back down through loop (back in hole)
5. Tighten
```

**Use**: Securing to trees, fixed anchor points

## Configuration #1: A-Frame (Classic)

**Setup**:
```
    Ridge Line
   ___________
  /           \
 /             \
```

**Steps**:
1. Tie ridgeline between two trees (6-7 feet high)
2. Drape tarp over ridgeline (centered)
3. Stake out corners at 45° angle
4. Adjust tension for taut pitch

**Pros**:
- Simplest setup
- Good rain protection
- Stable in wind
- Both ends open for ventilation

**Cons**:
- Limited headroom
- Both ends exposed to weather
- No floor protection

**Best For**: Quick overnight, good weather

## Configuration #2: Lean-To

**Setup**:
```
    |||||
   |\\\\\
   | \\\\\
   |  \\\\\
```

**Steps**:
1. Tie ridgeline between trees (5-6 feet high)
2. Hang tarp with one edge along ridgeline
3. Stake out opposite edge to ground
4. Stake sides at angle

**Pros**:
- Open front (fire warming, cooking view)
- Quick setup
- Good ventilation
- Easy access

**Cons**:
- One side completely exposed
- Less wind protection
- Rain can blow in

**Best For**: Fire camping, mild weather, scenic views

**Variations**:
- **High lean-to**: Ridgeline 7+ feet, more standing room
- **Low lean-to**: Ridgeline 3-4 feet, more wind protection

## Configuration #3: Diamond/Flying Diamond

**Setup**:
```
      /\
     /  \
    /    \
   /      \
```

**Steps**:
1. Tie ridgeline between trees
2. Orient tarp as diamond (corner up)
3. Attach top corner to ridgeline
4. Stake bottom corner to ground
5. Stake side corners out at 45°

**Pros**:
- Maximum coverage for tarp size
- Good rain shedding
- Decent wind protection
- Versatile

**Cons**:
- Less headroom than A-frame
- Requires more stakes/tie-outs

**Best For**: Solo shelter, variable weather

## Configuration #4: Wedge/Half-Pyramid

**Setup**:
```
    ____
   |    \
   |     \
   |______\
```

**Steps**:
1. Attach one edge to tree/pole (vertical)
2. Stake opposite corners out to form wedge
3. Stake remaining corner(s)

**Pros**:
- Three sides enclosed
- Good wind/rain protection
- Stable structure
- Better warmth retention

**Cons**:
- Less ventilation
- Can be stuffy
- Condensation in humid conditions

**Best For**: Cold weather, high wind, rain

## Configuration #5: C-Fly (Wind Shed)

**Setup**:
```
  ___
 /   \___
/        \
```

**Steps**:
1. One corner high on tree
2. Opposite corner staked low to ground
3. Creates curved profile
4. Side corners staked at middle height

**Pros**:
- Excellent wind shedding
- Aerodynamic shape
- Good rain protection
- Stable in storms

**Cons**:
- Complicated setup
- Less interior space
- Requires precise tensioning

**Best For**: High wind, exposed locations, storms

## Configuration #6: Adirondack (Three-Sided)

**Setup**:
```
  _______
 |\\\\\\\|
 | \\\\\\ |
 |  \\\\\ |
```

**Steps**:
1. Create rectangular frame with ridgeline and two upright poles
2. Drape tarp over frame
3. Back and sides enclosed
4. Front completely open

**Pros**:
- Maximum fire warming
- Standing room
- Good weather protection from three sides
- Social/group friendly

**Cons**:
- Requires poles or trees in right configuration
- More setup time
- One side exposed

**Best For**: Base camp, fire camping, group shelters

## Configuration #7: Plow Point

**Setup**:
```
      *
     /|\
    / | \
   /  |  \
```

**Steps**:
1. Tie center of tarp to single point (tree branch or pole)
2. Stake four corners out at angles
3. Creates pyramid shape

**Pros**:
- Single anchor point (one tree)
- Fully enclosed
- Very stable
- Good in 360° wind

**Cons**:
- Center pole takes floor space (if using pole)
- Less headroom at edges
- Can pool water at corners if not taut

**Best For**: Limited anchor points, variable wind direction

**Variations**:
- **High plow point**: More headroom, less protection
- **Low plow point**: Better warmth/protection, less room

## Configuration #8: Burrito/Tube Shelter

**Setup**:
```
  ========= (rolled sides)
 |         |
 |_________|
```

**Steps**:
1. Lay tarp flat on ground
2. Lie in center
3. Roll sides up over you
4. Secure with rocks, logs, or stakes

**Pros**:
- Emergency bivy
- Maximum warmth
- No setup needed
- Full protection

**Cons**:
- Claustrophobic
- Poor ventilation
- Condensation issues
- Emergency only

**Best For**: Extreme emergency, exhaustion, hypothermia risk

## Seasonal Adaptations

### Summer Setup

**Priorities**: Ventilation, shade, bug protection

- High pitch for air circulation
- Open sides for breeze
- Light colors reflect heat
- Elevate floor (airflow underneath)
- Bug netting at openings
- Lean-to or A-frame preferred

### Winter Setup

**Priorities**: Heat retention, wind blocking, snow load

- Low pitch (less wind, better heat retention)
- Enclosed sides (wedge, plow point)
- Dark colors absorb heat
- Steep angle sheds snow
- Insulated floor (boughs, mat)
- Add side walls with snow or debris
- Clear snow from around shelter

### Rain Setup

**Priorities**: Water shedding, dry interior

- Steep angle (min 45°)
- Ensure tarp doesn't touch ground (wicks water)
- Dig perimeter trench for runoff
- Extra guy lines for tension (prevents pooling)
- Seal seams if needed
- A-frame or diamond best

### Wind Setup

**Priorities**: Stability, aerodynamics

- Low profile (reduce wind resistance)
- Orient narrow end to wind
- Extra stakes and guy lines
- C-fly or wedge configuration
- Avoid flat surfaces (catch wind)
- Weight edges with rocks/logs

## Advanced Techniques

### Multi-Tarp Systems

**Layered Protection**:
- Upper tarp for rain (angled)
- Lower tarp for ground/wind
- Air gap between (insulation)

**Extended Coverage**:
- Connect multiple tarps
- Overlapping edges
- Create larger shelter or vestibule

### Ground Sheet Integration

**Methods**:
- Separate ground sheet under shelter
- "Bathtub" style (edges up sides)
- Tarp as both roof and floor (burrito)

**Sizing**:
- Ground sheet slightly smaller than overhead tarp
- Prevents rain channeling underneath
- Or tuck edges under sleeping pad

### Guy Line Optimization

**Placement**:
- Every grommet for max stability
- Minimum: corners + midpoints
- 45° angle to ground ideal
- Longer line = better angle = more strength

**Tensioning**:
- Use trucker's hitch for mechanical advantage
- Taut = better rain shedding
- Check and re-tension after rain (stretch)

### Anchor Points Without Trees

**Trekking Poles**:
- Adjustable height
- Lightweight
- Dual-purpose gear
- Use at corners or center

**Rock/Log Anchors**:
- Wrap line around heavy rocks
- Bury log/rock (deadman anchor)
- Weight tarp edges

**Stake Types**:
- Thin soil: long, thin stakes
- Hard ground: thick, strong stakes
- Sand/snow: wide stakes or buried bags (deadman)

## Field Modifications

### Leaky Tarp Repair

**Quick Fix**:
- Duct tape (dry surface)
- Pine pitch (natural sealant)
- Plastic bag patches
- Move setup to avoid leak area

**Long-Term**:
- Seam sealer
- Patch kits
- Silicone sealant

### Torn Grommet

**Bypass**:
- Tie line around entire corner
- Reinforce with duct tape
- Create new tie point with tape + washer

### Insufficient Cord

**Extend**:
- Join cords with sheet bend knot
- Use natural cordage (vines, roots)
- Rip cloth strips as cord
- Paracord inner strands

## Common Mistakes

**Over-Complicating**:
- Simple setups work best
- Practice basics first
- Save complex for specific conditions

**Poor Site Selection**:
- Setup quality doesn't matter if site floods
- Check drainage first
- Avoid dead tree hazards
- Consider sun/wind direction

**Insufficient Tension**:
- Loose = water pooling
- Loose = flapping in wind
- Tight = rain shedding

**Touching Ground**:
- Tarp edge on ground wicks water inside
- Keep 6"+ gap or trench around perimeter

**Wrong Configuration for Conditions**:
- Open lean-to in driving rain
- Enclosed wedge in summer heat
- Match setup to weather

## Practice Recommendations

**Skills to Master**:
1. Four basic setups (A-frame, lean-to, diamond, wedge)
2. Knots (trucker's hitch, taut-line, bowline)
3. Quick setup (under 5 minutes)
4. Setup in rain
5. Setup in dark
6. One-tree setups
7. No-tree setups

**Progression**:
- Backyard practice (unlimited time)
- Timed setups (build speed)
- Weather practice (rain/wind)
- Limited light/dark
- Different terrain
