# Teletext CSS Layout Clone

This is a clone of the Galax Teletext layout, built for local development and retro UI demos.

## Files

- `index.html` — main demo page
- `teletext.css` — core style sheet
- `teletext.js` — sizing and clock scripts
- `MODE7GX3.TTF` — **placeholder font** (you must add your own Teletext font here)

## Instructions

1. Download or supply a compatible MODE7 or Teletext font (for example `MODE7GX3.TTF`).
2. Place the font in the same folder as these files.
3. Open `index.html` in your browser.
4. Resize the window to see auto-scaling of the 40×25 character grid.

For the original inspiration and visual reference, see:  
<https://galax.xyz/TELETEXT/>
