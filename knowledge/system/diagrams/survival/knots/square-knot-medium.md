# Basic Knot Tying - Square Knot

**Purpose:** Join two ropes of equal thickness
**Uses:** First aid, bundling, general purpose
**Strength:** Medium (not for critical loads)
**Tier:** Medium (40×60)

---

## Overview

The square knot (also called reef knot) is one of the most fundamental knots for joining two ropes of similar diameter. While not suitable for critical loads, it's essential for everyday tasks in survival situations.

**Key Rule:** "Right over left, left over right makes the knot both tidy and tight"

---

## Step-by-Step Instructions

### Step 1: Cross Right Over Left

```
    Rope A          Rope B
      │               │
      │               │
      │      ┌────────┤
      │      │        │
      └──────┼────────┘
             │
           Cross
```

Take the right rope (A) and cross it over the left rope (B).

### Step 2: Wrap Around

```
    Rope A          Rope B
      │        ╱────┘
      │      ╱
      │    ╱
      └──<
         ╲
          ╲────┐
               │
        Pull through
```

Wrap rope A around rope B and pull the end through the loop.

### Step 3: Cross Left Over Right

```
    Rope A          Rope B
       │              │
    ┌──┴──┐           │
    │     └───────────┤
    │                 │
    └─────────────────┘
         Reverse
```

Now reverse the direction - take what's now on the left and cross it over the right.

### Step 4: Wrap Again

```
    Rope A     Rope B
      │      ╱─┘
      │    ╱
      └──<
         ╲
          ╲──┐
             │
      Pull through
```

Wrap around and pull through again, mirroring step 2.

### Step 5: Tighten

```
    Rope A          Rope B
      ▲               ▲
      │               │
    ┌─┴───────────────┴─┐
    │   ███████████████  │
    │   ▓▓▓▓KNOT▓▓▓▓▓▓  │
    │   ███████████████  │
    └─┬───────────────┬─┘
      │               │
      ▼               ▼
    Pull            Pull
```

Pull all four ends firmly to tighten the knot.

## Verification

### ✅ Correct Square Knot

- Both ends exit on the **same side**
- Knot lies flat
- Easy to untie after load
- Won't slip under tension

**Visual Check:**
```
  ────┐   ┌────
      │   │
      └─×─┘
      Same side ✓
```

### ❌ Granny Knot (WRONG)

- Ends exit on **opposite sides**
- Knot twists awkwardly
- Can slip under load
- Unreliable and dangerous

**Visual Check:**
```
  ────┐
      │ ×
      │   ┌────
      └───┘
      Different sides ✗
```

---

## Common Mistakes

### 1. Both Crossings Same Direction
**Problem:** Cross R→L, then R→L again (should be R→L, then L→R)
**Result:** Granny knot - unreliable and dangerous

### 2. Insufficient Tightening
**Problem:** Leaving knot loose
**Result:** Can slip apart under load
**Fix:** Pull all 4 ends firmly before use

### 3. Unequal Rope Thickness
**Problem:** Using ropes of different diameters
**Result:** Knot won't hold properly
**Fix:** Use sheet bend knot instead for unequal ropes

## Variations

### Surgeon's Knot (Extra Secure)

For added security, use a double wrap in the first crossing:

```
Step 1a: Double wrap
──────────────────
Rope A      Rope B
  │      ╱╱───┘
  │    ╱╱
  └──<<
     ╲╲
      ╲╲──┐
```

Then continue with normal square knot steps. This prevents the knot from loosening while you complete it.

## Survival Applications

### ✅ **Good Uses**

| Application | Description |
|-------------|-------------|
| **First Aid** | Secure bandages, tie slings, bundle splints |
| **Camp Setup** | Bundle firewood, tie gear together, temporary repairs |
| **Clothing** | Tie torn fabric, emergency belt, shoe repairs |

### ❌ **Never Use For**

- Climbing or rappelling
- Critical loads
- Life safety situations
- Securing heavy items

**Warning:** Always use proper climbing knots (figure-8, bowline) for life safety.

## Practice Guide

### Beginner Level
1. Use different colored ropes (easier to see crossings)
2. Practice 10 times slowly
3. Check each step carefully
4. Verify square vs granny every time

### Intermediate Level
1. Switch to same color ropes
2. Practice 20 times
3. Gradually increase speed
4. Try tying behind your back

### Advanced Level
1. Practice in complete darkness
2. Practice one-handed tying
3. Try with gloves on
4. Teach someone else (best way to master)

## Untying Methods

### Method 1: Push and Pull

Push the center of the knot while pulling one end:

```
       ↓
    ┌──▼──┐
    │Push │
    └──┬──┘
       │ Pull
       ▼
```

### Method 2: Twist and Loosen

Twist the knot to loosen, then pull apart:

```
       ↻
    ┌─────┐
    │Twist│
    └─────┘
```

## Safety Notes

⚠️ **CRITICAL WARNINGS:**

- **NOT for life safety** - Never use for climbing, rappelling, or any situation where failure could cause injury
- **Check regularly** - Inspect knot under continuous load
- **Replace damaged rope** - Never use frayed, cut, or weakened rope
- **Practice first** - Master this knot before relying on it in emergencies
- **Know your limits** - Understand rope strength and knot efficiency

---

## Rope Care Tips

### ✅ Do

- Inspect rope before each use
- Keep dry when possible (wet rope loses 30% strength)
- Store loosely coiled
- Protect from UV exposure and sharp edges

### ❌ Don't

- Use damaged or frayed rope
- Expose to chemicals (acids, solvents)
- Store wet (causes mildew and rot)
- Step on rope (grinds in dirt and damages fibers)
