## ‚ö†Ô∏è Critical Skills Checklist

Before relying on wilderness medicine knowledge, ensure you can:

- [ ] **Recognize life-threatening emergencies** (ABC: Airway, Breathing, Circulation)
- [ ] **Perform CPR** (30 compressions : 2 breaths, correct depth/rate)
- [ ] **Clear airway obstruction** (Heimlich maneuver, back blows)
- [ ] **Stop severe bleeding** (direct pressure, tourniquet if needed)
- [ ] **Identify shock** (pale, cold, rapid pulse, confusion)
- [ ] **Treat hypothermia** (gradual rewarming, no direct heat)
- [ ] **Splint fractures** (immobilize joint above and below)
- [ ] **Clean and dress wounds** (irrigation, sterile dressing)
- [ ] **Prevent infection** (hand washing, sterile technique)
- [ ] **Recognize when to evacuate** (beyond your skill level)

## üìã MARCH Protocol Priority

**Wilderness First Aid follows MARCH protocol:**

1. **M**assive Hemorrhage - Stop life-threatening bleeding first (1-2 min to death)
2. **A**irway - Clear airway obstruction (3-4 min to death)
3. **R**espiration - Treat breathing problems (5-10 min to serious harm)
4. **C**irculation - Address shock, fluid loss (15-30 min to serious harm)
5. **H**ypothermia/Head - Prevent heat loss, protect spine (1-3 hours to serious harm)

**See:** [[reference/first-aid-quick-reference]] for detailed protocols

## üè• Recommended Certifications

**These guides supplement, but don't replace, formal training:**

- **Wilderness First Aid** (WFA) - 16-hour course, basic wilderness medicine
- **Wilderness First Responder** (WFR) - 80-hour course, extended care
- **Wilderness EMT** (WEMT) - Advanced wilderness emergency care
- **Standard First Aid/CPR** - Essential baseline for everyone
- **Stop the Bleed** - Tourniquet and hemorrhage control training
