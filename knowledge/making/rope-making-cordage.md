# Rope Making & Natural Cordage

**Category:** Making - Practical Crafts
**Priority:** MEDIUM - Wilderness skill, self-reliance
**Season:** Year-round, easier when plants fresh
**Difficulty:** Intermediate - requires practice

---

## Overview

Making rope and cordage from natural materials is an ancient skill that provides crucial utility in survival situations and connects us to traditional crafts.

**Uses for natural rope:**
- Shelter construction
- Tool making
- Fishing line
- Snares/traps
- Lashing
- Bindings
- Emergency repairs

## Best Plants for Cordage (Australian)

### Excellent Choices

**Lomandra (Mat Rush):**
```
├─ Very common in Australia
├─ Long, flat leaves
├─ Very strong fibers
├─ Available year-round
├─ Leaves 30-100cm long
└─ Traditional Aboriginal use
```

**Preparation:**
- Cut mature leaves (avoid young, soft ones)
- Strip off edges (sharp!)
- Dry slightly (not fully)
- Separate into thinner strips if thick

**Strength:** Excellent - can support body weight

---

**Grass Tree (Xanthorrhoea):**
```
├─ Old flower stalks excellent
├─ Straight, long, strong
├─ Dry stalks best
├─ Can split for lashing
└─ Resin also useful (glue)
```

**Preparation:**
- Harvest dead flower stalks
- Clean off debris
- Split if needed
- Very strong as-is

**Strength:** Excellent - traditional spear making

---

**Native Flax (Phormi um):**
```
├─ Similar to New Zealand flax
├─ Long, strong fibers
├─ Found in gardens, wetlands
├─ Leaves 1-3m long
└─ Very versatile
```

**Preparation:**
- Cut mature leaves
- Scrape out soft pulp
- Left with strong fibers
- Can twist or braid

**Strength:** Excellent - traditional rope material

---

**Stringybark Eucalyptus:**
```
├─ Inner bark has fibers
├─ Strip bark in long pieces
├─ Separate layers
├─ Fine fibers inside
└─ Common across Australia
```

**Preparation:**
- Peel bark
- Soak if dry
- Separate into thin layers
- Twist fibers

**Strength:** Good - better for lighter cordage

---

### Good Choices

**Native Grasses (various species):**
- Long blades best
- Dry slightly (not fully brittle)
- Bundle multiple strands
- Weaker individually, strong bundled

**Sedges:**
- Wetland plants
- Long, triangular stems
- Split or use whole
- Good flexibility

**Pandanus (Screw Pine):**
- Coastal areas
- Long, strong leaves
- Sharp edges (be careful)
- Soak and split

## Basic Twisting Technique (2-Ply)

### The Reverse Wrap Method

**Step by step:**

```
1. Prepare fibers
   ├─ 2 bundles of fibers
   ├─ Each about pencil thickness
   ├─ Length: 60-100cm to start
   └─ Slightly damp (flexible, not wet)

2. Starting
   ├─ Hold both bundles together at one end
   ├─ Tie simple knot OR
   ├─ Loop one around the other
   └─ This is your anchor

3. The twist
   ├─ Hold anchor point between thumb and finger of left hand
   ├─ Take top bundle in right hand
   ├─ Twist it clockwise (away from you)
   ├─ While still twisted, wrap it COUNTER-clockwise over other bundle
   ├─ Hold that crossed position with left hand
   ├─ Now twist second bundle clockwise
   ├─ Wrap counter-clockwise over first
   └─ Repeat

4. The rhythm
   ├─ Twist, twist, twist bundle A
   ├─ Wrap over bundle B
   ├─ Hold
   ├─ Twist, twist, twist bundle B
   ├─ Wrap over bundle A
   ├─ Hold
   └─ Repeat down the length
```

**Visual pattern:**
```
Start:    A ║ B

Step 1:   A twisted clockwise
         (A) ║ B

Step 2:   A wrapped over B
          B ║(A)

Step 3:   B twisted clockwise
          B(║) A

Step 4:   B wrapped over A
         (B)║ A

Repeat...
```

**Key principles:**
- Always twist same direction (clockwise)
- Always wrap opposite direction (counter-clockwise)
- Keep tension even
- Tighter twist = stronger cord
- The wrap direction opposes the twist (keeps it from unraveling)

---

### Adding Length (Splicing)

**When bundle gets short:**

```
1. Before you run out (10-15cm left)
   └─ Plan ahead

2. Taper end
   ├─ Remove some fibers from end
   ├─ Creates gradual thinning
   └─ Not a blunt end

3. Add new bundle
   ├─ Also taper the starting end
   ├─ Overlap tapered sections (10-15cm)
   ├─ Twist together as you work
   └─ Should be seamless

4. Stagger splices
   ├─ Don't add to both bundles at same spot
   ├─ Alternate (add to A, work 20cm, then add to B)
   └─ Prevents weak points
```

**Result:** Continuous rope of any length

### 2-Ply Twist Technique (Visual)


### Cordage Construction Flowchart

```diagram flowchart
START: Gather plant fibers
STEP: Process fibers (ret or scrape) to remove pulp
STEP: Divide into bundles based on desired thickness
STEP: Choose technique: 2-ply twist (general use) or 3-ply braid (strength)
STEP: Twist individual bundles clockwise while wrapping counter-clockwise
STEP: Add new fibers by splicing (stagger joins for strength)
STEP: Maintain tension and evenness throughout
END: Finished cordage ready for use
```

## 3-Ply Braided Cordage

**Stronger and more flexible than 2-ply**

### Basic Braid

```
1. Start with 3 bundles
   ├─ Tie together at one end
   ├─ Pin/clamp anchor point
   └─ Spread bundles

2. Braid pattern
   ├─ Left bundle over center
   ├─ Right bundle over center
   ├─ Left bundle over center
   ├─ Right bundle over center
   └─ Repeat

3. Keep tight
   ├─ Pull firm after each cross
   ├─ Even tension
   └─ Consistent pattern
```

**Visual:**
```
      L C R        L over C         R over C
      │ │ │        │╱│ │            │ │╲│
      │ │ │   →    C L │      →     │ C R
      │ │ │        │ │ │            │ │ │
```

**Adding length:** Same as 2-ply, but one strand at a time

## Processing Plant Fibers

### Retting (For Bark/Stem Fibers)

**Purpose:** Separate strong fibers from soft material

```
1. Harvest
   ├─ Cut stems/bark
   ├─ Best when fresh
   └─ Mature plants (stronger)

2. Soak
   ├─ Submerge in water (creek, bucket)
   ├─ Weight down (rocks)
   ├─ Leave 1-2 weeks
   ├─ Bacteria breaks down soft tissue
   └─ Check regularly (smell = ready)

3. Rinse
   ├─ Remove from water
   ├─ Rinse in clean water
   └─ Remove loosened material

4. Separate fibers
   ├─ Beat gently (breaks down remaining pulp)
   ├─ Rub between hands
   ├─ Comb out fibers
   └─ Left with long, strong fibers

5. Dry
   ├─ Hang in shade (sun makes brittle)
   ├─ Air circulation
   ├─ Store dry
   └─ Dampen slightly before use
```

---

### Mechanical Extraction (For Leaf Fibers)

**For plants like lomandra, flax:**

```
1. Harvest leaves
   ├─ Cut mature leaves
   ├─ Avoid young, soft growth
   └─ Long as possible

2. Scrape
   ├─ Lay leaf on hard surface
   ├─ Use blunt knife or shell
   ├─ Scrape away soft pulp
   ├─ Work from base to tip
   ├─ Both sides
   └─ Left with fiber skeleton

3. Wash
   ├─ Rinse fibers
   ├─ Remove remaining pulp
   └─ Squeeze gently

4. Dry/use
   ├─ Can use damp (more flexible)
   ├─ Or dry for storage
   └─ Dampen before using if dried
```

## Applications

### Lashing (Tying Poles Together)

**Square lashing** (right angle):

```
1. Start with clove hitch on vertical pole
2. Wrap diagonally over cross
3. Under vertical, over horizontal
4. Around and around (6-8 wraps)
5. Wrap between poles (frapping)
6. Tie off with clove hitch
```

**Diagonal lashing** (X join):

```
1. Timber clove hitch around both poles
2. Wrap around one fork of X
3. Wrap around other fork
4. Frapping turns between
5. Tie off
```

---

### Fishing Line

**Requirements:**
- Thin but strong
- Smooth (won't spook fish)
- Can be single-ply if strong fiber

**Best approach:**
1. Select finest, strongest fibers
2. Tight 2-ply twist
3. Test before use (breaking strength)
4. Can add hook, sinker, lure

**Traditional hooks:**
- Carved bone
- Thorn
- Bent wire
- Carved wood

---

### Bow Drill Cordage

**For fire-starting bow:**

```
Requirements:
├─ About 1m length
├─ Moderate strength
├─ Some give (not too stiff)
└─ Thick enough to grip drill

Best materials:
├─ Stringybark inner bark
├─ Grass bundles
├─ Lomandra
└─ 2-ply construction sufficient
```

---

### Snare Line

**For trapping (where legal):**

```
Requirements:
├─ Strong (animal will pull)
├─ Smooth (won't snag on trigger)
├─ Appropriate diameter (animal size)
└─ Doesn't rot quickly

Construction:
├─ 2-ply minimum
├─ 3-ply better
├─ Test breaking strength
└─ Replace regularly
```

## Rope Strength Testing

**Always test before critical use:**

```
1. Visual inspection
   ├─ Even thickness
   ├─ No weak spots
   ├─ Tight twist
   └─ No frayed sections

2. Gradual loading
   ├─ Tie to secure point
   ├─ Pull gently
   ├─ Increase force gradually
   ├─ Feel for weak spots
   └─ Listen for cracking sounds

3. Breaking test (sample)
   ├─ Make test length
   ├─ Pull to breaking
   ├─ Note where it failed
   ├─ If failed at splice, redo splices
   └─ If failed at random point, technique needs work

4. Safe working load
   ├─ Breaking strength ÷ 5 = safe load
   ├─ Natural rope is variable
   ├─ Build in safety margin
   └─ Don't trust life to it (use modern rope)
```

## Storage & Maintenance

### Drying

```
├─ Hang in coils
├─ Shade (sun degrades fibers)
├─ Air circulation
├─ Fully dry before storing
└─ Mold ruins natural cordage
```

### Storage

```
├─ Cool, dry place
├─ Hung or coiled
├─ Away from rodents (they chew)
├─ Check periodically
└─ Can last years if dry
```

### Before use

```
├─ Inspect for damage
├─ Slightly dampen (increases flexibility)
├─ Don't soak (too weak when wet)
└─ Test strength
```

## Quick Expedient Cordage

**Need rope RIGHT NOW?**

### Clothing fibers:

```
├─ Cotton t-shirt: tear into strips, twist
├─ Jeans: pull threads from seam
├─ Shoelaces: already cordage!
└─ Strips of any fabric twisted together
```

### Plant materials (untreated):

```
├─ Long grass: bundle and twist (weak but fast)
├─ Green shoots: pliable, temporary binding
├─ Vines: some species usable as-is
└─ Roots: willow, some others flexible when fresh
```

```
├─ Long hair can be braided
├─ Surprisingly strong

└─ Emergency only
```

## Safety Notes

**Plant identification:**
- Know what you're harvesting
- Some plants irritate skin
- Avoid protected species
- Get permission on private land

**Harvesting sustainably:**
- Don't take all leaves from one plant
- Rotate harvesting locations
- Let plants recover
- Dead material when possible

**Testing:**
- Never rely on natural cordage for life safety without thorough testing
- Modern rope for climbing, rappelling, critical use
- Natural cordage excellent for low-stakes applications

## Troubleshooting

**Rope keeps untwisting:**
- Twist tighter
- Ensure wrap direction opposes twist
- Hold tension while working
- Practice rhythm

**Weak spots/breaks:**
- Uneven fiber thickness
- Bad splice (taper ends)
- Too dry (slightly damp)
- Poor quality fibers

**Can't make progress:**
- Fibers too dry (dampen)
- Fibers too wet (dry slightly)
- Not enough twist in bundles
- Inconsistent technique

**Lumpy, uneven:**
- Maintain consistent tension
- Check fiber distribution (even thickness)
- Practice, practice, practice
